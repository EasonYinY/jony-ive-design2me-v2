# 识别性设计增强 v2 (Identity Design Enhancement v2)

> **版本**: 2.0
> **日期**: 2026-06-21
> **触发**: Round 2 D2 折叠泵压识别性 8.5/10，距艾维级差 0.5 分
> **升级**: 从 旧版的"标志性形态+细节+材料对比"增强为"识别性锚点+多层识别+品类突破"

---

## 旧版 问题

Round 2 D2 发现：
- 折叠杠杆识别性 8.5/10，距艾维级（≥9/10）差 0.5 分
- 折叠机构在咖啡机品类中罕见，但"折叠"本身不是独特识别性
- 需要更强的"一眼识别"特征

**核心问题**: 识别性依赖于"罕见形态"，但罕见≠记忆点。需要创造"无法忘记"的视觉锚点。

---

## 旧版 增强策略

### 策略1: 识别性锚点法

**识别性锚点**: 一个极端特征，用户看到后立即关联到产品。

| 锚点类型 | 示例 | 效果 |
|---------|------|------|
| 极端比例 | 40:1 螺旋柱 | 一眼记住"那个超细的咖啡机" |
| 极端材料对比 | 镜面不锈钢 + 粗糙混凝土 | 一眼记住"金属与石头的咖啡机" |
| 极端形态 | 双螺旋DNA结构 | 一眼记住"DNA咖啡机" |
| 极端颜色 | 全黑 + 唯一红色按钮 | 一眼记住"红点咖啡机" |
| 极端纹理 | 全光滑 + 唯一粗糙区域 | 一眼记住"有疤的咖啡机" |

**规则**: 每个方向必须有≥1个识别性锚点，且锚点必须"极端"到无法忘记。

### 策略2: 多层识别法

识别 → 识别：

```
Layer 1: 远距离识别（5米外）
  → 整体轮廓、极端比例、主色调
  
Layer 2: 中距离识别（1米外）
  → 标志性形态、材料对比、关键特征
  
Layer 3: 近距离识别（0.3米内）
  → 精密细节、纹理、工艺痕迹
```

**示例**（D3双螺旋）:
```
Layer 1: 5米外 → "那个超细的发光柱子"（4.0:1修长比例）
Layer 2: 1米外 → "DNA双螺旋咖啡机"（双螺旋结构）
Layer 3: 0.3米内 → "铜节点精密配合"（0.2mm间隙阴影）
```

### 策略3: 品类突破法

**问题**: 在品类中"罕见"不够，需要"突破品类认知"。

| 策略 | 示例 | 效果 |
|------|------|------|
| 形态突破 | 咖啡机不是圆柱/方形，而是螺旋/折叠/悬浮 | 突破"咖啡机=圆柱"认知 |
| 材料突破 | 咖啡机不是金属/塑料，而是石头/玻璃/木材 | 突破"咖啡机=金属"认知 |
| 交互突破 | 咖啡机不是按钮/开关，而是旋转/滑动/感应 | 突破"咖啡机=按钮"认知 |
| 姿态突破 | 咖啡机不是放置，而是悬浮/悬挂/嵌入 | 突破"咖啡机=放置"认知 |

**规则**: 至少在一个维度上突破品类认知，创造"这不是咖啡机"的惊讶感。

## 旧版 识别性评分标准

| 分数 | 标准 | 描述 |
|------|------|------|
| 10/10 | 全球唯一 | 品类中从未出现，无法混淆 |
| 9/10 | 品类突破 | 突破品类认知，一眼识别 |
| 8/10 | 品类罕见 | 品类中罕见，但可联想 |
| 7/10 | 有特征 | 有识别特征，但不够强烈 |
| ≤6/10 | 无识别性 | 与品类原型混淆 |

**艾维级要求**: ≥9/10（品类突破）

---

## 旧版 自检清单

在 STEP-10 推导形态后，检查识别性：

```
[ ] 有≥1个识别性锚点（极端到无法忘记）
[ ] 有三层识别（远/中/近）
[ ] 至少在一个维度突破品类认知
[ ] 远距离（5米）可识别轮廓
[ ] 中距离（1米）可识别形态
[ ] 近距离（0.3米）可识别细节
[ ] 无"类似XX""参考XX"等依赖描述
```

## 与 旧版 对比

| 维度 | 旧版 | 升级版 | 效果 |
|------|------|------|------|
| 识别层数 | 1层 | 3层 | 从单一→丰富 |
| 锚点强度 | "罕见" | "极端" | 从罕见→无法忘记 |
| 品类关系 | 在品类中定位 | 突破品类认知 | 从定位→突破 |
| 评分标准 | ≥8/10 | ≥9/10 | 从准艾维→艾维级 |
| 自检 | 3项 | 7项 | 更严格 |
