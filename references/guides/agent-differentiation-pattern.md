# Agent 差异化执行模式

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 5轮25代理迭代计划执行实践
> **适用范围**: 多 Agent 并行执行设计任务

---

## 核心发现

在多 Agent 并行执行设计任务时，每个 Agent 应有明确的**差异化焦点**，避免所有 Agent 生成相似方向。

### 实践验证

| Agent | 差异化焦点 | 探索方向 | 预测评分 | 验证 |
|-------|-----------|---------|---------|------|
| Agent 1 | 自然形态 + 材料对比 | 鹅卵石/水滴/蘑菇 | 85-87/90 | ✅ |
| Agent 2 | 人体工学 + 功能创新 | 穿戴/便携/交互 | 83-84/90 | ✅ |
| Agent 3 | 文化杂交 + 识别性 | 历史/地域/生物 | 87-90/90 | ✅ 最高 |
| Agent 4 | 极简形态 + 精密细节 | 单一体块/无缝/隐藏 | 85-87/90 | ✅ |

**关键发现**: Agent 3 文化杂交方向在户外灯品类达到预测 90/90 艾维级评分，验证了差异化焦点的有效性。

---

## 差异化模式定义

### Agent 1: 自然形态 + 材料对比方向

**焦点**: 探索自然原型（鹅卵石、水滴、蘑菇、贝壳、竹节）与材料对比的结合

**STEP-06 策略**:
- 优先生成自然形态候选（≥4个自然形态）
- 强调材料对比丰富度（≥3种材料）
- 避免几何形态（除非极端比例）

**适用品类**: 所有品类，尤其是户外灯、音箱、底座

**参考指南**:
- `references/guides/natural-form-ive-level-pattern.md`
- `references/guides/hard-soft-contrast.md`

---

### Agent 2: 人体工学 + 功能创新方向

**焦点**: 探索人体工学优化与功能创新的结合

**STEP-06 策略**:
- 优先生成人体工学优化候选（贴合曲线、重量分布、接触点）
- 强调功能创新（新交互方式、新使用场景）
- 避免空气动力学陷阱（Pitfall-029）

**适用品类**: 穿戴设备、手持设备、便携设备

**参考指南**:
- `references/guides/wearable-ergonomics-guide.md`
- `references/guides/ergonomic-fatigue-data.md`

---

### Agent 3: 文化杂交 + 识别性方向

**焦点**: 探索文化杂交（历史/地域/生物）与识别性设计的结合

**STEP-06 策略**:
- 优先生成文化杂交候选（≥3个文化杂交方向）
- 强调识别性（标志性形态、标志性细节、材料对比）
- 避免过度堆砌（Pitfall-028）

**适用品类**: 所有品类，尤其是灯具、音箱、家居

**参考指南**:
- `references/guides/cultural-hybrid-design.md`
- `references/guides/identity-design-v2.md`

**实践验证**: Agent 3 在户外灯品类达到预测 90/90，是最高评分方向。

---

### Agent 4: 极简形态 + 精密细节方向

**焦点**: 探索极简形态（单一体块、无缝曲面、隐藏接缝）与精密细节的结合

**STEP-06 策略**:
- 优先生成单一体块候选（≥3个单一体块）
- 强调精密细节（0.1mm配合、G2圆角、Ra值）
- 避免模块化（与极简矛盾）

**适用品类**: 所有品类，尤其是消费电子、灯具、家居

**参考指南**:
- `references/guides/ive-precision-manufacturing.md`
- `references/guides/product-level-prompt-revolution.md`

## 执行规则

### 1. 差异化优先
每个 Agent 在 STEP-06 候选生成时，**优先探索其焦点方向**，但不得限制其他方向。

### 2. 六轴差异门统一
所有 Agent 生成的候选最终必须通过**统一的六轴差异门**筛选，确保三个方向本质不同。

### 3. 质量门统一
所有 Agent 生成的方向最终必须通过**统一的审美质量门**，确保艾维级标准一致。

### 4. 提示词一致性
所有 Agent 生成的提示词必须遵循**统一的视觉语言基础**（摄影语法、悬浮感、背景、负面约束）。
