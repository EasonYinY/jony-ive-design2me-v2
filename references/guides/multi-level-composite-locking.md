# 多层级复合体块锁定与异质材质职责链(Stub)

> **⚠️ 本文件为占位 stub,正式内容待补充。**

## 1. 协议意图

精密设备(无人机、显微镜、机器人臂、咖啡机)经常呈现"主体外壳 + 旋转关节 + 精密端头"三段式结构。

**核心原则**:
- **宏观**: 形态张力、CMF 品牌一致性
- **关节**: 旋转/铰接/对焦 — 异质材质(金属关节 + 树脂减重件)显性化功能
- **精密端**: 微距/接触/夹持 — 高硬度材料(陶瓷/钛合金),细节放大到 0.1mm 精度

**禁止**: 三段用同一种材质 + 同一种表面处理 = 视觉扁平化 + 功能不可读。

## 2. 触发条件

- 设备含 ≥ 2 个旋转/滑动/铰接关节
- 设备有"宏观体块 + 端头操作"结构(如无人机臂 + 云台;咖啡机主体 + 出水口)
- STEP-09 六轴差异门需要"结构"维度差异化

## 3. 最小运行骨架

```
1. 拆分为 3 层: 宏观层 / 关节层 / 精密端
2. 每层指派唯一材质 + 唯一表面处理(避免 3 层同质)
3. 关节层必须用 异质材质(高对比度,与上下两层不同)
4. 精密端细节 = 0.1mm 精度级(刻度/接缝/倒角/指示)
5. STEP-11 交互节点只出现在关节层 + 精密端(不在宏观层)
```
