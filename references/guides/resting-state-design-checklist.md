# 静息状态设计检查清单

> **版本**: 1.0
> **日期**: 2026-06-22
> **触发**: Round 1评审发现静息状态是艾维级设计的分水岭（蘑菇9/10=79分 vs 北欧火炬6/10=64分）
> **适用范围**: 所有产品设计任务，目标艾维级（≥80/90）

---

## 核心原则

艾维的设计哲学：**"产品在关闭时也是美的"**（The product is beautiful even when it's off）

Round 1实证：
- 静息状态9/10 → 总分79-83/90（艾维级/准艾维级高分）
- 静息状态7/10 → 总分69-75/90（准艾维级中低分）
- 静息状态6/10 → 总分64/90（普通级）

**关键发现**: 静息状态每提升1分，总分提升约3-5分

---

## 静息状态三要素

### 要素1: 放置稳定性

**定义**: 产品在任意放置姿态下保持稳定，不滚动、不倾倒

| 检查项 | 标准 | 优化方法 |
|--------|------|----------|
| 底部设计 | 底面直径≥60mm或配重底座 | 不锈钢配重+硅胶防滑 |
| 重心位置 | 重心偏下（低于几何中心） | 底部电池仓+金属配重 |
| 防滑处理 | 底部硅胶/橡胶接触面 | 液态硅胶包覆底部 |
| 多角度放置 | 支持平放/倾斜/悬挂 | 微凸底座+隐藏吊环 |

**常见失败**:
- 禅石平板（18mm厚）：极端扁平，只能"捏持"，放置稳定但握持困难
- 北欧火炬：长条形，平放时滚动，竖放时不稳定
- 水滴：倾斜放置需底部配重，否则易倾倒

### 要素2: 姿态优雅

**定义**: 产品静置时如自然物般优雅，姿态符合形态逻辑

| 检查项 | 标准 | 优化方法 |
|--------|------|----------|
| 形态逻辑 | 放置姿态符合自然原型 | 鹅卵石→平放如静石；蘑菇→直立如森林蘑菇 |
| 视觉平衡 | 形态重心与视觉重心一致 | 对称或 purposeful 不对称 |
| 姿态选择 | 提供≥2种优雅放置姿态 | 平放+倾斜/悬挂 |
| 无工具感 | 静置时不显露"设备"属性 | 隐藏接口、隐藏按钮 |

**成功案例**:
- 鹅卵石：平放如静石，自然倾斜15°展示最佳视角
- 蘑菇：垂直放置如森林蘑菇，帽部如伞覆盖
- 水滴：自然倾斜，重心低，姿态流动

**失败案例**:
- 折叠臂：折叠后体积大，静息状态不优雅
- 竹叶：卷曲形态更"动态"而非"静穆"
- 北欧火炬：长条形，静息时无"自然物"美感

### 要素3: 悬浮感

**定义**: 产品似有似无地悬浮于表面，减少与支撑面的"沉重接触"

| 检查项 | 标准 | 优化方法 |
|--------|------|----------|
| 底座设计 | 微凸底座（接触面最小化） | 底部微凸硅胶垫（直径20mm） |
| 阴影控制 | 阴影柔和、边缘模糊 | 底部微凸减少阴影面积 |
| 视觉轻盈 | 形态不"压"在表面上 | 底部收窄+顶部舒展 |
| 悬挂状态 | 悬挂时自然下垂，不僵硬 | 隐藏吊环+柔软挂绳 |

**成功案例**:
- 鹅卵石：微凸底部，放置如"浮于桌面"
- 蘑菇：底座微凸，直立如"生长于表面"
- 水滴：底部配重+上部轻盈，"锚定-悬浮"张力

## 静息状态评分标准

| 评分 | 标准 | 示例 |
|------|------|------|
| 10/10 | 任意放置均优雅，如艺术品般静穆 | 鹅卵石平放如禅石 |
| 9/10 | 主要放置姿态优雅，次要姿态可接受 | 蘑菇直立+悬挂 |
| 8/10 | 放置稳定但姿态一般，略有工具感 | 工业方块平放 |
| 7/10 | 放置基本稳定，但姿态不优雅 | 竹叶自立但动态感强 |
| 6/10 | 放置不稳定或姿态差 | 北欧火炬平放滚动 |
| ≤5/10 | 放置困难，无静息美学 | 球形无底座，任意滚动 |

## 检查清单

### 设计前检查

```
[ ] 产品是否支持≥2种放置姿态？
[ ] 底部是否有防滑/配重设计？
[ ] 重心是否偏下？
[ ] 放置姿态是否符合自然原型逻辑？
[ ] 是否隐藏了接口/按钮等设备元素？
[ ] 底座是否为微凸设计（减少接触面积）？
[ ] 悬挂状态是否自然（若适用）？
```

### 设计后检查

```
[ ] 平放时是否稳定不滚动？
[ ] 姿态是否如自然物般优雅？
[ ] 关闭时是否仍是美的？
[ ] 是否有"悬浮感"？
[ ] 是否避免了"工具感"？
[ ] 静息状态评分是否≥9/10？
```

## 提示词增强模板

在提示词中增加以下描述以强化静息状态：

```
静息状态设计:
- 底部微凸硅胶垫（直径20mm，高2mm），放置如浮于表面
- 重心偏下（底部配重不锈钢，重心位于几何中心下方5mm）
- 支持平放（自然倾斜15°）和悬挂（隐藏吊环）双模式
- 关闭时作为静物仍有美学价值，如自然物般静置
- 隐藏所有接口和按钮，表面纯净无干扰
```
