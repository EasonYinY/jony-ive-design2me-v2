# 用户设计方法论

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 用户反馈设计"太寡淡或太乱""无亮点""纹理乱"，并显式传授其设计方法论

---

## 概述

本指南记录用户（Eason）的设计方法论，作为技能内置方法论的补充。当用户说"按我的方法"或类似指令时，优先使用本方法论。

---

## 设计推导顺序（5层）

```
第1层：大造型/大感觉/大风格
  → 先抓住整体轮廓、体量感、风格定位
  → 这是设计的"魂"，必须先定

第2层：部件造型与轮廓关系
  → 每个部件的造型
  → 部件之间的关系：穿插/倒角/咬合/组合/连接/桥接
  → 比例关系、节奏感

第3层：部件CMF
  → 每个部件的材料、颜色、表面处理
  → 材料之间的对比和呼应

第4层：特殊细节特征
  → 网格、纹理、刻度、灯光
  → 点睛之笔：小点、小凸点、小凹点、丝印图标

第5层：画面整体感觉
  → 灯光、背景、环境
  → 当前统一要求纯白背景
```

---

## 设计原则

### 1. 有轻有重，有显性有隐性

- **显性焦点**：第一眼看到的，大比例差异、强烈材质对比、特殊形态
- **隐性细节**：第二眼发现的，精致刻度、微凹点、隐藏式灯光
- 不能全显性（太乱），也不能全隐性（太寡淡）

### 2. 不能太寡淡，也不能太乱太复杂

- **寡淡**：无亮点、无对比、无细节、无重点
- **太乱**：纹理杂乱、元素堆砌、找不到重点
- 要有"呼吸感"，有疏有密

### 3. 要有亮点/重点/点睛之笔

每个设计必须有至少1个让人"哇"的点：
- 可能是形态上的突破
- 可能是材质对比的惊喜
- 可能是细节处理的精致

### 4. 惊喜时刻/暖心时刻

| 类型 | 示例 |
|------|------|
| 品牌标识 | 微型logo、丝印图标、精致刻度 |
| 触觉细节 | 小凸点、小凹点、纹理变化 |
| 灯光设计 | 发丝级LED、渐变灯光、隐藏式灯光 |
| 特殊材质 | 局部金属镶嵌、透明窗口、磁性吸附 |
| 特殊工艺 | 激光蚀刻、微喷砂、阳极氧化渐变 |

---

## 部件关系类型

| 关系 | 描述 | 适用场景 |
|------|------|----------|
| 穿插 | 一部件穿过另一部件 | 创造动态感 |
| 倒角 | 边缘倒角过渡 | 柔和过渡 |
| 咬合 | 一部件咬入另一部件 | 精密连接 |
| 组合 | 多个部件组合成整体 | 模块化 |
| 连接 | 部件之间通过连接件连接 | 可拆卸 |
| 桥接 | 一部件桥接两个部件 | 结构支撑 |

## 应用示例

### 电动剃须刀（应用本方法论）

```
第1层：大造型
→ 整体修长悬浮感，流线型轮廓，未来科技风格

第2层：部件关系
→ 手柄主体与刀头模块通过磁悬浮间隙桥接
→ 刀头偏置15度，与手柄形成非对称动态比例
→ 穿插关系：LED灯带穿插于手柄缝隙中

第3层：部件CMF
→ 手柄：拉丝铝，微喷砂，深空灰（主材）
→ 刀头：镍镀层，镜面，银白色（对比材）
→ 间隙：磁体场，无直接接触（隐性材质）

第4层：细节特征
→ 侧面触控条：发丝级LED指示灯（显性焦点）
→ 刀头网面：精密蚀刻网格，0.1毫米间隙
→ 底部：微型品牌刻字（隐性细节，点睛之笔）
→ 手柄纹理： Directional micro-blast pattern（触感细节）

第5层：画面整体
→ Floats. Neutral gradient. Pure white background.
```
