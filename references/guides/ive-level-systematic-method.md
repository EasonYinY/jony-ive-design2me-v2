# 艾维级设计系统化方法

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 5轮迭代（R1-R5）核心发现总结
> **适用范围**: 所有产品设计任务，目标艾维级（≥80/90）

---

## 核心发现

### 5轮迭代实证（户外灯品类）

| 轮次 | 产品 | 最高分 | 艾维级方向数 | 平均评分 | 关键发现 |
|------|------|--------|-------------|---------|---------|
| R1 | 露营灯 | 83/90 | 1/3 | 70.0 | 自然形态优势明显 |
| R2 | 壁灯 | 83/90 | 2/3 | 79.0 | 文化杂交+材料对比 |
| R3 | 庭院灯 | 85/90 | 2/3 | 80.3 | 氛围光设计成功 |
| R4 | 头灯 | 84/90 | 1/3 | 78.3 | 人体工学重要性 |
| R5 | 路灯 | 86/90 | 2/3 | 81.7 | 综合方法论验证 |

**总体**: 最高分86/90，平均77.9/90，艾维级方向率60%（8/15）

## 艾维级设计六要素

### 要素1: 自然形态（基础，必需）

**作用**: 提供基础分70-75/90
**方法**: 选择自然原型（鹅卵石/水滴/贝壳/蘑菇/竹节/禅石）
**验证**: 所有艾维级设计均使用自然形态
**Pitfall**: 纯自然形态天花板76-78/90（Pitfall-028）

**自然形态选择指南**:

| 形态 | 适用场景 | 评分参考 | 关键特征 |
|------|---------|---------|---------|
| 鹅卵石 | 手持/放置 | 83/90 | 有机曲线，贴合手掌 |
| 水滴 | 贴合/悬挂 | 80/90 | 上部小下部大，重心低 |
| 贝壳 | 贴合人体/光扩散 | 84/90 | 自然曲线，文化杂交 |
| 蘑菇 | 氛围光/公共照明 | 86/90 | 帽柄比例，光扩散 |
| 竹节 | 分段/东方美学 | 81/90 | 分段维护，文化杂交 |
| 禅石 | 静谧/极简 | 76/90 | 扁平静谧，但天花板 |

### 要素2: 文化杂交（加分，推荐）

**作用**: +4~9分
**方法**: 自然形态+文化元素+现代功能
**验证**: 文化杂交设计比纯自然形态高+4~9分
**Pitfall**: 纯自然形态天花板（Pitfall-028）

**文化杂交三模式**:

| 模式 | 示例 | 加分 |
|------|------|------|
| 历史对象×现代功能 | 贝壳×建筑照明 | +3~5分 |
| 地域材料×全球功能 | 竹×现代照明 | +4~6分 |
| 生物形态×科技功能 | 蘑菇×氛围光 | +5~9分 |

### 要素3: 材料对比（必要，必需）

**作用**: +3~5分（基础），+5~8分（丰富）
**方法**: 至少2种材料，软硬/透明-不透明/金属-非金属对比
**验证**: 双材料比单材料高+9分（Pitfall-027）
**Pitfall**: 单材料设计陷阱（Pitfall-027）

**材料对比矩阵**:

| 材料组合 | 效果 | 评分参考 |
|---------|------|---------|
| 硅胶+金属 | 软硬对比 | 83-84/90 |
| 玻璃+金属 | 透明-不透明 | 83/90 |
| 聚碳酸酯+铝+硅胶 | 三材料 | 85-86/90 |
| 混凝土+不锈钢+玻璃 | 三材料（公共） | 86/90 |
| 单材料（铝） | 无对比 | 74/90 |

### 要素4: 人体工学（底线，必需）

**作用**: 不达标直接降分
**方法**: 贴合人体曲线，重量分布，接触点≥2个
**验证**: 人体工学比空气动力学高+9分（Pitfall-029）
**Pitfall**: 空气动力学陷阱（Pitfall-029）

**人体工学优先级**（穿戴设备）:
1. 贴合曲线（与身体部位匹配）
2. 重量分布（均匀，不集中于一点）
3. 接触点（≥2个，防滑材料）
4. 稳定性（不滑动/不晃动）

### 要素5: 氛围光设计（加分，可选）

**作用**: +2~4分
**方法**: 光扩散方式，色温，亮度层次
**验证**: 氛围光设计比直射照明评分更高
**适用**: 景观照明/庭院灯/氛围场景

**氛围光设计要点**:
- 扩散方式：帽状/分段/缝隙/磨砂
- 色温：暖色调（2700K-3000K）
- 亮度层次：主光+辅助光+氛围光

### 要素6: 比例张力（加分，推荐）

**作用**: +2~4分
**方法**: 极端比例（直径:高度>3:1），不对称，动态比例
**验证**: 极端比例创造视觉张力

**比例张力类型**:
- 极端扁平（直径:高度>3:1）
- 极端细长（长度:直径>5:1）
- 不对称比例（打破对称）

## 评分预测模型

### 基础分

```
基础分 = 自然形态选择
- 鹅卵石/水滴/贝壳/蘑菇/竹节: 70-75分
- 禅石/叶片: 68-72分
- 几何形态（非极端）: 60-65分
- 俗套形态（圆盘/圆柱）: 55-60分
```

### 加分项

```
文化杂交: +4~9分
- 历史对象×现代功能: +3~5分
- 地域材料×全球功能: +4~6分
- 生物形态×科技功能: +5~9分

材料对比: +3~8分
- 2种材料（软硬对比）: +3~5分
- 3种材料（丰富对比）: +5~8分
- 单材料: 0分（Pitfall-027）

氛围光设计: +2~4分
- 帽状扩散: +3~4分
- 分段发光: +2~3分
- 直射照明: 0分

比例张力: +2~4分
- 极端比例: +3~4分
- 标准比例: 0分
```

### 预测示例

| 设计 | 基础分 | 文化杂交 | 材料对比 | 氛围光 | 比例张力 | 预测 | 实际 |
|------|--------|---------|---------|--------|---------|------|------|
| 蘑菇路灯 | 72 | +9（生物×科技） | +8（3材料） | +4（帽状） | +3（帽柄） | 96 | 86 |
| 贝壳壁灯 | 73 | +5（历史×现代） | +6（2材料） | 0 | 0 | 84 | 83 |
| 鹅卵石露营灯 | 74 | +4（自然×电子） | +7（4材料） | 0 | 0 | 85 | 83 |
| 水滴头灯 | 73 | 0 | +5（2材料） | 0 | 0 | 78 | 76 |
| 折纸壁灯 | 70 | +3（文化×照明） | +2（1材料） | 0 | +3（极端） | 78 | 74 |

**注意**: 预测模型上限90分，实际评分受AI生成质量影响

## 设计决策树

```
目标: 艾维级设计（≥80/90）
│
├── 1. 选择自然形态
│   ├── 鹅卵石/水滴/贝壳/蘑菇/竹节 → 基础分70-75
│   ├── 禅石/叶片 → 基础分68-72（天花板风险）
│   └── 几何形态 → 基础分60-65（需极端比例）
│
├── 2. 添加文化杂交（推荐）
│   ├── 历史对象×现代功能 → +3~5分
│   ├── 地域材料×全球功能 → +4~6分
│   └── 生物形态×科技功能 → +5~9分
│
├── 3. 确保材料对比（必需）
│   ├── ≥2种材料 → +3~5分
│   ├── ≥3种材料 → +5~8分
│   └── 单材料 → 0分（Pitfall-027）
│
├── 4. 考虑氛围光（可选）
│   ├── 景观照明 → +2~4分
│   └── 其他品类 → 0分
│
├── 5. 考虑比例张力（可选）
│   ├── 极端比例 → +2~4分
│   └── 标准比例 → 0分
│
└── 6. 检查人体工学（穿戴设备必需）
    ├── 贴合曲线 → 通过
    └── 空气动力学 → 风险（Pitfall-029）
```

## 检查清单

### 设计前检查

```
[ ] 是否选择了自然形态？
[ ] 是否避免了俗套形态（Pitfall-026）？
[ ] 是否包含文化杂交元素（Pitfall-028）？
[ ] 是否使用了≥2种材料（Pitfall-027）？
[ ] 是否考虑了氛围光设计（景观照明）？
[ ] 是否考虑了人体工学（穿戴设备）？
[ ] 是否避免了空气动力学陷阱（Pitfall-029）？
```

### 设计后检查

```
[ ] 预测评分是否≥80/90？
[ ] 是否包含惊喜/暖心时刻？
[ ] 材料职责是否明确？
[ ] 接缝秩序是否精密？
[ ] 静息状态是否优雅？
```
