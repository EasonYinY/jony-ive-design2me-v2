> **版本**: 1.0
> **日期**: 2026-06-22
> **触发**: Round 2发现所有方向使用"触摸调光"，操作维度同质化

## 操作类型清单

### 触摸类

| 操作 | 描述 | 适用场景 | 人体工学 |
|------|------|---------|---------|
| 轻触 | 轻触表面调光 | 平滑表面 | 指尖 |
| 长按 | 长按开关机 | 任何表面 | 指腹 |
| 双击 | 双击切换模式 | 任何表面 | 指尖 |

### 旋转类

| 操作 | 描述 | 适用场景 | 人体工学 |
|------|------|---------|---------|
| 旋钮 | 旋转机械旋钮 | 顶部/侧面 | 拇指+食指 |
| 转盘 | 旋转转盘调光 | 顶部 | 单指 |
| 滚轮 | 滚动滚轮调光 | 侧面 | 拇指 |

### 滑动类

| 操作 | 描述 | 适用场景 | 人体工学 |
|------|------|---------|---------|
| 滑块 | 滑动滑块调光 | 侧面 | 拇指 |
| 触摸条 | 滑动触摸条 | 侧面 | 单指 |
| 拨杆 | 拨动拨杆开关 | 侧面 | 拇指 |

### 按压类

| 操作 | 描述 | 适用场景 | 人体工学 |
|------|------|---------|---------|
| 按钮 | 按压按钮开关 | 顶部/侧面 | 拇指 |
| 按键 | 按压按键调光 | 侧面 | 拇指 |
| 踏板 | 脚踏开关 | 底部 | 脚掌 |

### 握持类

| 操作 | 描述 | 适用场景 | 人体工学 |

| 倾斜感应 | 倾斜角度调光 | 整体 | 手腕 |

## 代理差异化分配

| 代理 | 操作类型 | 惊喜时刻 | 适用方向 |
|------|---------|---------|---------|
| Agent 1 | 触摸 | 微凸触控区 | 禅石/鹅卵石 |
| Agent 2 | 旋转 | 旋钮阻尼感 | 冰川/水滴 |
| Agent 3 | 滑动 | 滑块轨道 | 蘑菇/帽柄 |
| Agent 4 | 按压 | 按压反馈 | 火山/锥形 |
| Agent 5 | 握持 | 握持感应 | 叶片/珊瑚 |

## 检查清单

### 设计前检查

```
[ ] 每个代理的操作类型是否不同？
[ ] 操作类型是否与方向形态匹配？
[ ] 操作的人体工学是否合理？
[ ] 惊喜时刻是否与操作类型相关？
```

### 设计后检查

```
[ ] 3方向的操作是否本质不同？
[ ] 操作是否易于理解？
[ ] 操作是否与功能匹配？
[ ] 预测评分是否≥80/90？
```
