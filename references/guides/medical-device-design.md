# 医疗器械设计硬约束指南(II 类手持电动)

> **日期**: 2026-06-24
> **触发**: 案例 027 电动吻合器任务实证;识别出 II 类手持电动手术器械的 recurring 硬约束模式
> **适用范围**: 胃肠/肺/肝/血管等开放/腔镜手术的 II 类手持电动器械(吻合器/电刀/超声刀/缝合器)

## 七项硬事实(II 类手持电动器械通用)

执行 STEP-04 时,以下 7 项事实必须建立(全部可迁移):

| 事实 | 数值 | 来源类型 | 依据 |
|---|---|---|---|
| 手柄握持直径 | 40-55mm | 人体测量 | 外科医生虎口宽 90-110mm(男 95% 置信) |
| 手柄长度 | 130-180mm | 功能需求 | 容纳 1-2 节 18650 + 马达 + 控制板 |
| 主操作键行程 | 18-25mm | 工程约束 | 18mm 提供足够触觉确认,25mm 极限 |
| 副控制(指环/旋钮)内径 | 18-22mm | 人体测量 | 男性拇指近端指节 |
| 手柄重量上限 | 280-400g | 工程约束 | 8h 长时握持疲劳指数 < 30 |
| 末端执行器击发力 | 80-200N(视器械) | 工程约束 | 钛钉/切割/缝合所需最低推力 |
| 灭菌温度上限 | 134°C × 4min × 2000 次 | 法规 | ISO 17665 蒸汽灭菌 |

## 矛盾三角 recurring pattern(STEP-05 核心)

常见主矛盾结构 = **动力 + 人体 + 法规**三者同时约束:

```
动力 = 80-200N 末端击发(功能不可让步)
人体 = ≤ 400g 重量 + ≤ 25mm 行程 + 8h 疲劳 < 30(医生不可让步)
法规 = 134°C 高压蒸汽灭菌 2000 次 + II 类注册(II 类不可让步)
```

**唯一常见解法**:电动直驱(主操作键仅作开关,马达直接推动末端)

**被否决路径**:
- 机械杠杆/齿轮增力:减主操作力但重量超 400g
- 微型液压:液体泄漏 + 灭菌难,违反 FDA 21 CFR 888
- 气压/气动:需外部气源,不可便携

## 法规清单(II 类手术器械,STEP-07 必查)

| 法规 | 适用范围 | 关键约束 |
|---|---|---|
| ISO 13485:2016 | 质量管理体系 | 设计控制 + 风险管理 + 可追溯性 |
| IEC 60601-1:2005+A1:2012 | 医用电气设备通用安全 | 电气安全 + 漏电流 + 接地 |
| ISO 17665-1:2006 | 蒸汽灭菌 | 134°C × 4min × 2000 次循环 |
| FDA 21 CFR 878.4750 | 植入式吻合钉 | II 类(US)/III 类(部分永久植入) |
| FDA 21 CFR 888 | 骨科/外科器械 | 内部无液体要求 |
| ISO 14971:2019 | 风险管理 | 风险分析 + 风险控制 + 残余风险评估 |
| IEC 60601-1-2 | 电磁兼容 | 抗扰度 + 发射 |
| ISO 10993-1 | 生物相容性 | 直接/间接接触患者材料评估 |

## 文化杂交 × 医疗器械(REF-GUIDE-009 §类型 1 在医疗场景的可行杂交)

`REF-GUIDE-009` §三种文化杂交类型中,在医疗场景里**类型 1(历史对象 × 现代功能)**最稳定:

| 历史对象 | 提取的结构原理 | 现代功能映射 | 案例 |
|---|---|---|---|
| 东亚砚台 | 厚重 slab + 凹陷盛水 + 边角修圆 | 充电坞底座(承重 + 接口 + 防刮) | D3 双足 slab 底座(2026-06-24) |
| 青铜鼎 | 三足鼎立 + 声学腔体 | 智能医疗设备(三足稳定 + 传感腔) | 失败案例,三足成本高 |
| 怀表 | 表盘刻度 + 蓝钢指针 | 仪表显示 + 状态指示 | 状态指示灯设计 |
| 黄铜望远镜 | 套筒式 + 同轴调焦 | 模块化器械(可换刀头) | 模块化吻合器 |

**否决条件**:
- 文化元素只影响装饰(如砚台纹样贴花)→ 不通过
- 文化元素与功能冲突(如鼎的香炉功能 → 消毒后无意义)→ 不通过
- 西方医院/手术室接受度未调研 → HYPOTHESIS 标记

## brief 字数预判(医疗器械类)

医疗类 brief 通常 12-25 字,几乎必触发 `REF-RULE-015`:

| 字符数 | 阈值 | 必备 unconfirmed 项 |
|---|---|---|

## 应用模板

### STEP-04 模板(医疗器械版)

```markdown
## 第一性物理与人体事实(II 类手持电动 [器械类型])

1. 手柄握持直径 = [40-55mm] ← 人体测量:外科医生虎口宽 90-110mm
2. 手柄长度 = [130-180mm] ← 功能需求:容纳 2×18650 + 马达 + 控制板
3. 主操作键行程 = [18-25mm] ← 工程约束:18mm 触觉确认,25mm 极限
4. 副控制内径 = [18-22mm] ← 人体测量:男性拇指近端指节
5. 手柄重量 = [280-400g] ← 工程约束:8h 长时握持疲劳指数 < 30
6. 末端击发力 = [80-200N] ← 工程约束:[具体组织] [具体钉/刀] 测算
7. 灭菌温度 = [134°C × 4min × 2000 次] ← 法规:ISO 17665

## 法规清单
- ISO 13485:2016
- IEC 60601-1:2005+A1:2012
- FDA 21 CFR 878.4750 (US)
- ISO 14971:2019 风险管理
```

### STEP-05 模板(矛盾三角)

```markdown
## 核心矛盾(II 类手持电动 [器械类型])

**主矛盾**:在 ≤ [重量]g 手柄重量 + ≤ [行程]mm 主操作键行程的硬约束下,
既要实现 [击发力]N 末端执行器推力(动力),
又要让 8h 累计疲劳指数从 > 60 降至 < 30(人体),
还必须保持 II 类医疗器械的高压蒸汽灭菌兼容性(法规)。

**解法**:电动直驱(主操作键仅作开关,马达直接推动末端)
**否决路径**:机械杠杆(重量超)/微型液压(法规)/气压(便携性)
```
