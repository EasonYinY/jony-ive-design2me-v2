# 功能识别性强化方法

> **版本**: 1.0
> **日期**: 2026-06-22
> **触发**: Round 1评审发现禅石(66/90)的"平板"形态被误认为"充电宝"，功能识别性弱
> **适用范围**: 极简形态设计、目标艾维级（≥80/90）

---

## 核心原则

**艾维的设计中"功能识别性"从不妥协**：如AirPods一看就是耳机，iPhone一看就是手机。

Round 1实证：
- 蘑菇（帽状光扩散）: 3秒识别，79/90
- 鹅卵石（360°光扩散）: 3秒识别，83/90
- 禅石（平板+边缘发光）: 需解释，66/90
- 工业方块（发光面板）: 基本识别，74/90

**关键发现**: 功能识别性每降低1级，总分降低约5-8分

---

## 功能识别性层级

### 层级1: 即时识别（3秒内）

**定义**: 用户3秒内能准确识别产品类型和核心功能

| 设计 | 识别特征 | 评分参考 |
|------|----------|---------|
| 蘑菇 | 帽状光扩散罩 = 灯 | 79/90 |
| 鹅卵石 | 360°磨砂发光 = 灯 | 83/90 |
| 水滴 | 上部收敛+下部发光 = 灯 | 76/90 |

### 层级2: 延迟识别（3-10秒）

**定义**: 用户需观察细节后才能识别产品类型

| 设计 | 识别障碍 | 评分参考 |
|------|----------|---------|
| 工业方块 | 方块形态更像"充电宝/硬盘" | 74/90 |
| 蜂巢 | 球体+六边形更像"足球/分子模型" | 71/90 |
| 竹叶 | 叶片形态更像"装饰品" | 70/90 |

### 层级3: 难以识别（>10秒或需解释）

**定义**: 用户无法独立识别产品类型，需要文字说明

| 设计 | 识别障碍 | 评分参考 |
|------|----------|---------|
| 禅石 | 平板形态 = "充电宝/硬盘/电子秤" | 66/90 |
| 精密胶囊 | 胶囊形态 = "音箱/加湿器" | 75/90 |

## 强化方法

### 方法1: 核心功能形态暗示

**原则**: 形态本身必须暗示"发光""扩散""调节"等核心功能

| 功能 | 形态暗示 | 示例 |
|------|----------|------|
| 发光/照明 | 穹顶/帽状/球状光扩散面 | 蘑菇帽、鹅卵石磨砂外壳 |
| 定向照明 | 收敛/聚焦形态 | 水滴上部收敛 |
| 氛围光 | 柔和曲线+半透明材料 | 磨砂玻璃/聚碳酸酯 |
| 便携/手持 | 握持直径40-60mm | 柄部、中段握持区 |
| 悬挂 | 隐藏吊环/顶部挂孔 | 水滴隐藏吊环 |

### 方法2: 微凸功能标识

**原则**: 通过微型凸起/凹陷暗示功能，不破坏整体形态

| 功能标识 | 尺寸 | 位置 | 效果 |
|----------|------|------|------|
| 微凸发光环 | 直径80mm，凸出2mm | 正面中心 | 暗示"发光区域" |
| 微凹触控区 | 直径15mm，凹陷1mm | 顶部 | 暗示"操作区域" |
| 微凸调节环 | 宽度5mm，凸出1mm | 柄部中段 | 暗示"旋转调节" |
| 微缝指示灯 | 长度10mm，宽度0.5mm | 底部边缘 | 暗示"电量/状态" |

### 方法3: 光扩散可视化

**原则**: 即使产品关闭，光扩散结构也应可见

| 方法 | 描述 | 示例 |
|------|------|------|
| 磨砂半透明外壳 | 外壳本身暗示"光从内部扩散" | 磨砂聚碳酸酯 |
| 发光面边界 | 明确的发光面与主体边界 | 玻璃扩散环嵌入主体 |
| 光扩散纹理 | 同心圆/渐变纹理暗示光路 | 帽部内侧同心圆纹理 |
| 缝隙发光 | 边缘缝隙暗示"光从缝隙溢出" | 禅石边缘0.3mm缝隙 |

### 方法4: 操作直觉性

**原则**: 用户看到产品即知道如何操作

| 操作 | 直觉设计 | 示例 |
|------|----------|------|
| 开关 | 顶部微凸区域 = 按压 | 鹅卵石顶部触控区 |
| 调光 | 侧面微凹条 = 滑动 | 水滴侧面触控条 |
| 旋转 | 柄部调节环 = 旋转 | 蘑菇柄部硅胶环 |
| 悬挂 | 顶部隐藏吊环 = 拉出 | 水滴隐藏吊环 |
| 充电 | 底部微型接口 = 磁吸 | 鹅卵石底部磁吸充电 |

## 极简形态的特殊处理

极简形态（禅石/平板/方块）容易陷入"功能识别性弱"陷阱，需额外强化：

### 策略1: 增加"光扩散穹顶"

**原设计**: 极端扁平平板（18mm厚）
**优化**: 增加微凸穹顶（中心凸出8mm，直径60mm）
**效果**: 穹顶暗示"光扩散"，同时保持极简美学

### 策略2: 增加"发光边界环"

**原设计**: 隐藏式发光边缘
**优化**: 正面微凸发光环（直径80mm，凸出2mm）
**效果**: 关闭时作为"装饰性精密圈"，开启时作为"光扩散边界"

### 策略3: 增加"操作触点"

**原设计**: 隐藏式触控边缘
**优化**: 侧面微凹握持区（深度2mm，长度40mm）+ 硅胶条
**效果**: 握持区暗示"手持设备"，同时增强人体工学

## 检查清单

### 设计前检查

```
[ ] 3秒内是否能识别产品类型？
[ ] 形态是否暗示核心功能（发光/扩散/调节）？
[ ] 是否包含微凸/微凹功能标识？
[ ] 光扩散结构是否可见（即使关闭）？
[ ] 操作方式是否直觉（无需说明书）？
```

### 设计后检查

```
[ ] 用户测试：3秒内识别率是否≥80%？
[ ] 是否避免了"像X而非灯"的评价？
[ ] 功能标识是否破坏整体美学？
[ ] 极简形态是否牺牲了功能识别性？
[ ] 是否可增加微凸环/穹顶等强化识别？
```

## 提示词增强模板

在提示词中增加以下描述以强化功能识别性：

```
功能识别性强化:
- 核心识别符: [形态特征]明确暗示"这是灯"
- 光扩散暗示: [磨砂半透明外壳/帽状穹顶/边缘缝隙]暗示发光功能
- 操作直觉: [顶部微凸=按压/侧面微凹=滑动/柄部环=旋转]
- 微凸功能标识: [尺寸+位置+功能]
- 3秒识别: 用户3秒内能识别这是"便携式照明设备"
```
