# 艾维级精密制造细节规范

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 用户反馈"设计太飘逸、质感不顶级、缺少细节"
> **目标**: 从"概念描述"升级到"工程图纸级"提示词

---

## 核心问题

当前提示词的问题：
1. **设计太飘逸**: 形态模糊，边缘不清晰
2. **质感不顶级**: 材料像塑料玩具，缺少重量感
3. **缺少细节**: 表面空白，无功能结构

---

## 艾维级精密制造六要素

### 1. 几何精确度（必须）

| 参数 | 规范 | 示例 |
|------|------|------|
| 尺寸 | 精确到0.1mm | 长100.0mm，宽70.0mm，高45.0mm |
| 圆角 | G2连续，标注R值 | 边缘G2圆角R3.0mm |
| 拔模角度 | 标注角度 | 底部拔模角度1.5° |
| 壁厚 | 标注厚度 | 壁厚2.5mm |
| 配合间隙 | 标注间隙 | 精密配合间隙0.1mm |

### 2. 制造精度描述（必须）

| 工艺 | 描述规范 | 示例 |
|------|---------|------|
| CNC | 刀纹间距、表面粗糙度 | 5轴CNC，刀纹间距0.5mm，Ra 0.8 |
| 阳极氧化 | 膜厚、颜色、光泽 | 阳极氧化，膜厚25μm，深空灰，哑光 |
| 注塑 | 纹理、粗糙度 | 精密注塑，细磨砂，Ra 1.6 |
| 喷砂 | 粒度 | 喷砂粒度#240 |
| 拉丝 | 方向、间距 | 拉丝纹理，间距0.3mm |

### 3. 功能细节清单（必须）

| 功能 | 描述规范 | 示例 |
|------|---------|------|
| 接口 | 类型+防水等级 | USB-C接口，硅胶防水塞，IP67 |
| 指示灯 | 数量+尺寸+位置 | 4颗LED电量指示灯，微孔直径1mm，侧面 |
| 按键 | 类型+尺寸+触感 | 硅胶按键，直径12mm，行程0.5mm，微动开关 |
| 散热 | 结构+尺寸 | 散热鳍片，间距2mm，高度5mm |
| 防水 | 结构+等级 | 硅胶密封圈，IP67 |
| 磁吸 | 力值+材料 | 磁吸底座，吸力5N，钕铁硼磁体 |

### 4. 材料质感描述（必须）

| 质感 | 描述规范 | 示例 |
|------|---------|------|
| 重量感 | 重心+配重 | 重心偏下，不锈钢配重盘，直径60mm，厚8mm |
| 温度感 | 材料导热性 | 铝合金导热，表面温度均匀 |
| 触感纹理 | 粗糙度+纹理 | 防滑纹理，Ra 3.2，细磨砂 |
| 光泽度 | 哑光/高光/拉丝 | 哑光阳极氧化，拉丝不锈钢 |

### 5. 接缝秩序规范（必须）

| 接缝类型 | 描述规范 | 示例 |
|---------|---------|------|
| 精密配合 | 间隙值 | 0.1mm精密配合间隙 |
| 隐藏式接缝 | 位置+方式 | 隐藏式接缝，底部卡扣连接 |
| 过渡圆角 | R值 | 过渡圆角R2.0mm |
| 无可见接缝 | 工艺 | 双料注塑，无可见接缝（0.05mm配合） |

### 6. 静息状态表达（必须）

| 状态 | 描述规范 | 示例 |
|------|---------|------|
| 放置稳定性 | 重心+支撑 | 重心偏下，三点支撑，稳定放置 |
| 悬浮感 | 支撑方式 | 底部微凸，边缘悬浮0.5mm |
| 姿态 | 角度+方向 | 自然倾斜15°，展示最佳视角 |

## 提示词升级对比

### 当前提示词（问题版）

```
主体是暖白色聚碳酸酯，精密注塑，温润
```

**问题**:
- "精密注塑"太笼统
- "温润"太抽象
- 无尺寸、无纹理、无功能细节

### 升级后提示词（艾维版）

```
主体是暖白色聚碳酸酯，精密注塑，壁厚2.5mm，表面细磨砂（Ra 1.6），边缘G2圆角R3mm，底部拔模角度1.5°。顶部集成硅胶防水塞（USB-C接口，IP67），侧面隐藏式电量指示灯（4颗LED，微孔直径1mm）。底部不锈钢配重盘（CNC加工，直径60mm，厚8mm，拉丝纹理），确保重心偏下，稳定放置。
```

**升级点**:
- 尺寸精确（壁厚2.5mm）
- 纹理量化（Ra 1.6）
- 圆角标注（G2圆角R3mm）
- 功能细节（USB-C防水塞、电量指示灯）
- 制造精度（CNC、拉丝纹理）
- 重量感（不锈钢配重盘、重心偏下）

## 检查清单

### 提示词生成前检查

```
[ ] 是否包含精确尺寸（精确到0.1mm）？
[ ] 是否包含圆角标注（G2连续，R值）？
[ ] 是否包含制造精度（Ra值、刀纹、膜厚）？
[ ] 是否包含功能细节（接口、按键、指示灯）？
[ ] 是否包含材料质感（重量感、温度感、纹理）？
[ ] 是否包含接缝秩序（间隙、配合、隐藏式）？
[ ] 是否包含静息状态（重心、支撑、姿态）？
```

### 提示词生成后检查

```
[ ] 删除所有抽象形容词（"温润""精致""优雅"）
[ ] 替换为量化描述（"Ra 1.6""R3mm""壁厚2.5mm"）
[ ] 确保功能细节≥3个
[ ] 确保材料种类≥2种
[ ] 确保制造精度描述≥2处
```
