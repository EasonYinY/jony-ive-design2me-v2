# 概念来源库

## 强制要求

至少一个方向必须来自本库（非传统产品形态）。

## 概念-功能映射规则

1. **形态→结构**：从概念提取核心形态，映射到产品结构
2. **结构→功能**：验证结构是否能完成核心功能（剃须）
3. **功能→材质**：根据概念选择材质（冰川=透明/冷色，火山=流动/暖色）
4. **材质→工艺**：根据材质选择工艺（晶体=切割，液态=铸造）

## 否决条件

- 概念过于具象（像玩具/摆件）→ 不通过
- 概念与功能冲突（无法剃须）→ 不通过
- 概念来源无法识别 → 不通过
- 概念只是表面装饰（不影响结构）→ 不通过
