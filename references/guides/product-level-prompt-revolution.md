## 原"产品级革命"路线硬约束(已 SUPERSEDED · 仅作历史快照)

### 原原则 1: 零抽象形容词

**原"产品级革命"路线禁止使用的词**(已废弃):
- 温润、精致、优雅、高级、简约、大气
- 美观、漂亮、好看、时尚、潮流
- 质感好、手感好、视觉效果好

**本版本新规则**:
- **保留必要的感官形容词**:"haunting"(氛围光)、"serene"(禅意)、"dynamic"(动态比例)等是允许的,只要它们服务于视觉描述
- **禁止抽象形容词**:"高级金属""美观大方"等 → 替换为具体感官描述("hairline-brushed titanium slab""matte black with cool white indicator")

### 原原则 2: 强制工程参数(已废弃,改 本版本新规则)

**原"产品级革命"路线强制参数**(实测无效):

| 参数类型 | 原"产品级革命"路线要求 | 本版本实际规则 |
|---------|--------------|----------------|
| 精确尺寸 | ≥ 3 个,精确到 0.1mm | ≤ 2 个,精确到 ≥ 1mm 粗数字 |
| 壁厚 | ≥ 1 个,0.1mm 精度 | 不要求 |
| 圆角 | ≥ 1 个 G2 R3.0mm | 可选,G2 fillet / chamfer / tangent 是允许的 |
| 表面粗糙度 | ≥ 1 个 Ra 1.6 | 不要求,改用感官描述("hairline-brushed""matte""polished") |
| 材料牌号 | ≥ 1 个 6061-T6 | 不要求,改用感官描述("titanium slab""brass ring") |

### 原原则 3: 强制功能细节(已废弃,改方法 15 形态暗示)

**原"产品级革命"路线强制功能细节**(实测 Nano Banana Pro 渲染不了):
- USB-C 接口、硅胶防水塞、IP67
- 4 颗 LED 电量指示灯、微孔直径 1.0mm
- 硅胶按键、直径 12mm、行程 0.5mm
- 散热鳍片、间距 2mm、高度 5mm
- 不锈钢配重盘、直径 60mm、厚 8mm
- 硅胶密封圈、IP67

**本版本新规则**(方法 15 形态暗示):
- 删功能描述(rotary/slide/press/touch/scroll),用形态暗示
- 旋钮 → flush circular depression
- 按键 → small rectangular recess
- 接口 → small rectangular recess, no visible pins
- 指示灯 → small circular dot, cool white 5000K(从 prompt-word-bank § 5 选)

## 产品级提示词三原则

### 原则1: 零抽象形容词

**禁止使用的词**:
- 温润、精致、优雅、高级、简约、大气
- 美观、漂亮、好看、时尚、潮流
- 质感好、手感好、视觉效果好

**必须替换为**:
- 具体尺寸（长100.0mm）
- 表面粗糙度（Ra 1.6）
- 圆角半径（R3.0mm）
- 壁厚（2.5mm）
- 材料牌号（6061-T6铝合金）

### 原则2: 强制工程参数

**必须包含的参数**:

| 参数类型 | 数量要求 | 示例 |
|---------|---------|------|
| 精确尺寸 | ≥3个 | 长100.0mm，宽70.0mm，高45.0mm |
| 壁厚 | ≥1个 | 壁厚2.5mm |
| 圆角 | ≥1个 | 边缘G2圆角R3.0mm |
| 表面粗糙度 | ≥1个 | 表面细磨砂Ra 1.6 |
| 制造精度 | ≥2处 | 5轴CNC，刀纹间距0.5mm |
| 材料牌号 | ≥1个 | 6061-T6铝合金 |

### 原则3: 强制功能细节

**必须包含的功能细节**:

| 功能类型 | 数量要求 | 示例 |
|---------|---------|------|
| 接口 | ≥1个 | USB-C接口，硅胶防水塞，IP67 |
| 指示灯 | ≥1个 | 4颗LED电量指示灯，微孔直径1.0mm |
| 按键/开关 | ≥1个 | 硅胶按键，直径12mm，行程0.5mm |
| 散热结构 | ≥1个 | 散热鳍片，间距2mm，高度5mm |
| 固定/支撑 | ≥1个 | 不锈钢配重盘，直径60mm，厚8mm |
| 防水结构 | ≥1个 | 硅胶密封圈，IP67 |

## 产品级提示词模板

### 模板结构

```
[摄影语法] + [精确尺寸] + [材料工程参数] + [制造精度] + [功能细节] + [CMF对比] + [静息状态]
```

### 模板示例

```
工作室摄影，略高于四分之三视角。

[精确尺寸]
一个鹅卵石形态的户外露营灯，长100.0mm，宽70.0mm，高45.0mm，2.2:1.5:1有机比例。

[材料工程参数]
主体是6061-T6铝合金，5轴CNC加工，壁厚2.5mm，表面细磨砂Ra 1.6，边缘G2圆角R3.0mm，底部拔模角度1.5°。阳极氧化处理，膜厚25μm，深空灰色，哑光。

[功能细节]
顶部集成硅胶防水塞（USB-C接口，IP67，塞子直径8.0mm，高5.0mm）。侧面隐藏式电量指示灯（4颗LED，微孔直径1.0mm，间距3.0mm）。底部不锈钢配重盘（SUS304，CNC加工，直径60.0mm，厚8.0mm，拉丝纹理，刀纹间距0.3mm）。

[制造精度]
发光面是康宁大猩猩玻璃，热弯成型，壁厚3.0mm，边缘精密配合（0.1mm间隙），与主体卡扣连接。玻璃表面微纹理（喷砂粒度#240），均匀透光。

[CMF对比]
硅胶包覆是液态硅胶（LSR），双料注塑，厚度1.5mm，表面微纹理（防滑Ra 3.2），与铝合金主体无可见接缝（0.05mm配合）。

[静息状态]
产品放置于白色大理石台面，重心偏下（配重盘确保），三点支撑，稳定不倾斜。底部边缘微悬浮0.5mm。

[背景]
纯白背景，无环境，无人物。无logo，无文字，无螺丝。

[整体描述]
大造型：扁平鹅卵石，有机曲线与精密制造对比。部件关系：硅胶包覆主体+玻璃发光面+不锈钢配重盘+防水塞+电量指示灯。CMF：铝合金（深空灰磨砂）+液态硅胶（岩灰防滑）+大猩猩玻璃（透明透光）+不锈钢（拉丝配重），四材料诚实对比。细节：USB-C防水塞（硅胶，IP67），电量指示灯（4颗微孔LED），不锈钢拉丝配重盘（CNC，直径60mm），玻璃边缘精密配合（0.1mm），硅胶表面防滑纹理（Ra 3.2）。画面整体：自然形态，精密制造，功能细节丰富，重量感强，艾维级设计。
```

## 检查清单

### 生成前检查（强制）

```
□ 零抽象形容词检查：删除所有"温润""精致""优雅"等词
□ 精确尺寸检查：≥3个精确尺寸（精确到0.1mm）
□ 壁厚检查：≥1个壁厚参数
□ 圆角检查：≥1个圆角参数（G2连续，R值）
□ 表面粗糙度检查：≥1个Ra值
□ 制造精度检查：≥2处制造精度描述
□ 材料牌号检查：≥1个材料牌号
□ 功能细节检查：≥5个功能细节（接口、指示灯、按键、散热、固定、防水）
□ 重量感检查：≥1处重量感描述（重心、配重、密度）
□ 静息状态检查：≥1处静息状态描述（支撑、姿态、悬浮）
```

```

□ 确保材料牌号≥1个
□ 确保重量感描述≥1处
```

## 升级对比


```
长100.0mm，宽70.0mm，高45.0mm，壁厚2.5mm，边缘G2圆角R3.0mm，底部拔模角度1.5°，表面细磨砂Ra 1.6。顶部USB-C防水塞（IP67），侧面4颗LED电量指示灯（微孔直径1.0mm），底部不锈钢配重盘（直径60.0mm，厚8.0mm，拉丝纹理）。
```
