# 功能分区（Functional Zoning）— 复杂产品的硬约束

> **版本**: 1.0
> **日期**: 2026-06-23
> **触发**: 用户反馈"中规中矩，很普通"，并明确诊断"功能比较复杂的产品需要做功能的分区，有多个功能区来组成，每个区域都要有内部的结构、功能"——这是 STEP-06 候选生成 + STEP-10 形态推导的根因错位。

## 核心规则

**任何功能数 ≥ 3 的产品，必须先做功能区拆解，再选基础体块。** 直接套"1 个基础体块 = 1 个方向"是 simple product 模式，对 complex product 是失败的——会把 5 个功能区压扁成 1 块塑料壳。

### 何时触发本指南

| 产品复杂度 | 功能数 | 是否触发 | 模式 |
|---|---|---|---|
| Simple | 1-2 | ❌ | 1 个基础体块足够（如鼠标、剃须刀） |
| Medium | 3-4 | ⚠️ | 可选——主功能区 + 1-2 个辅助区 |
| **Complex** | **5+** | **✅ 必触发** | **必须按本指南做功能区拆解** |
| 复杂咖啡机 | 5（水箱/加热/操作/出水/接水）+ 1（电源/排气） | ✅ | 6 区，必须拆解 |
| 复杂头戴设备 | 6（显示×2+音频×2+计算+电池） | ✅ | 6 区 |
| 复杂无人机 | 5（机臂×4+中央仓） | ✅ | 5 区 |

## 流程修正：从 STEP-06 改起

### 原 STEP-06（错位）

```
STEP-06 候选生成：
  5 个候选 = 5 个整体造型（slab / cylinder / block / wedge / dome）
  → 失败原因：每个候选都是 1 个基础体块，5 个功能区被压扁
```

### 新 STEP-06（正确）

```
STEP-06.0 功能区拆解 [NEW]：
  - 列出 4-7 个功能区
  - 每个功能区定义：位置 / 物理边界 / 与其他区的接口 / 是否可见
  - 例（咖啡机）：
    1. 水箱区（底部或后部，800-1500ml）
    2. 加热区（中部，304 不锈钢锅炉 9 bar）
    3. 操作区（前部，启动点 + 反馈）
    4. 出水区（前下，Ø25-28mm 金属嘴）
    5. 接水区（前下，不锈钢杯托 + 排水）
    6. 电源/排气区（后部/底部，隐蔽）

STEP-06.1 候选方向生成（改）：
  - 每个候选 = 1 种**功能区排布方案**（不是 1 种基础体块）
  - 例：
    - CD1: 竖直分层（水箱底 + 加热中 + 操作顶 + 出水前下）
    - CD2: 水平分布（操作左 + 加热右 + 水箱后 + 出水前）
    - CD3: 中央放射（加热中心 + 其他环绕）

STEP-06.2 基础体块分配 [NEW]：
  - 在选定的功能区排布下，每个功能区独立选基础体块
  - 例（CD1 竖直分层）：
    - 水箱区 = cylinder（垂直圆筒）
    - 加热区 = cylinder Ø95mm
    - 操作区 = 平面 panel
    - 出水区 = nozzle（圆柱短嘴）
    - 接水区 = recessed tray
  - 区间连接 = 12mm G1 chamfer + 0.5mm parting line

STEP-10 形态推导（改）：
  - 不再是"1 个基础体块 + 变形规则"
  - 改为"5 个功能区各自的体块 + 区间接缝信号 + 整体层次比例"
```

## 艾维真实作品的功能区数（参考）

| 作品 | 功能区数 | 视觉信号 |
|---|---|---|
| iMac G4 (2002) | 5（半球底+杆+屏+音箱+接口） | 直径突变 12× |
| Linn Sondek LP12 (1973+) | 6（底板+主轴+唱臂+压片+马达+电源） | 5 件可见分件 |
| B&O Beosound 1 | 4（高频+中频+低频反射+触控） | 同径不同材质密度 |
| BALMUDA Sailing Lantern | 3（罩+灯芯+底座） | 透明罩暴露灯芯 |
| Ferrari Luce | 4（四圆灯+车身+轮+内饰） | 四圆灯重复信号 |
| Apple Pro Display XDR | 4（屏+背板+格栅+接口） | 格栅=视觉识别 |

**共同点**：每个艾维作品都 ≥ 4 个功能区，且**每个功能区都有自己的视觉信号**（不是压成 1 块）。

## DesignIR 新增必填字段（当前版本 建议）

在现有 15 字段基础上，新增 3 个：

| 字段 | 中文 | 完整性要求 |
|---|---|---|
| `functional_zones` | 功能区列表 | ≥ 3 个功能区，每个写明名称、位置、尺寸、与相邻区接缝 |
| `interior_visibility` | 内部结构可见性 | 哪些功能区可见（半透明/外露/可拆）、哪些隐藏 |
| `inter_zone_seams` | 区间接缝信号 | 每两个功能区之间的接缝类型（chamfer / parting line / material switch / gasket / light strip / magnetic line） |

## 视觉张力比例新定义（当前版本 建议）

**旧版 定义**：整体轮廓的极端比例（高径比 2.6:1 等）
**当前 定义**：**功能区之间的尺寸突变比**

| 作品 | 区-区尺寸比 | 张力来源 |
|---|---|---|
| iMac G4 | 底座 Ø250 → 杆 Ø30（突变 8.3×） | 细杆奇迹感 |
| Beosound 1 | 高频 Ø40 → 低频反射 Ø150（突变 3.75×） | 声学分层 |
| Linn LP12 | 底板 450 → 主轴 15（突变 30×） | 精密机械 |
| 目标咖啡机 | 出水嘴 Ø25 → 加热锅炉 Ø95（突变 3.8×） | 功能反差 |

**对每个方向**至少标注 1 个**功能区之间的尺寸突变比 ≥ 3×**——这是"艾维级视觉张力"的硬指标。

## STEP-14 质量门新增第 12 项

**质量门 12 — 功能区表达检查**：
- [ ] 设计是否表达了 ≥ 3 个功能区（视觉上可识别）？
- [ ] 是否 ≥ 1 个功能区半透明/外露（"诚实结构"）？
- [ ] 是否 ≥ 1 对功能区之间有 ≥ 3× 尺寸突变（视觉张力）？
- [ ] 每个功能区是否有独立材料或表面处理（材料对比）？

任一不通过 → 退回 STEP-10 重新推导。

## Round 2 实测（咖啡机 6 张图反查）

| 方向 | 功能区数 | 视觉信号 | 距艾维差距 |
|---|---|---|---|
| D1 桌面咖啡雕塑 | 1 区（被压扁） | 玻璃罩+洞石=2 块材料 | -3 分（功能压扁） |
| D2 三足仪式器物 | 2 区（勉强） | 圆柱+三足=2 段 | -2 分（信号弱） |
| D3 密闭器皿 | 2 区（勉强） | 块状+磁吸=2 段 | -2 分（抽屉被画成打开） |

**根因**：3 方向都被压扁成 1-2 区，没有按 5 区咖啡机标准做拆解。

## 提示词新模板（强制）

**旧版 模板**（单一段落）：
```
[基础体块] [整体尺寸] [单材料] [接触点] [负面约束] [背景]
```

**当前 模板**（分段功能区描述）：
```
[功能区 1: 位置 + 材料 + 尺寸 + 可见性]
[功能区 2: 位置 + 材料 + 尺寸 + 可见性]
[功能区 3: 位置 + 材料 + 尺寸 + 可见性]
[功能区 4: 位置 + 材料 + 尺寸 + 可见性]
[功能区 5: 位置 + 材料 + 尺寸 + 可见性]
[区间接缝信号: G1 chamfer 12mm / 0.5mm parting line / material switch / 磁力线]
[关键尺寸突变: 区-区 Ø 比 ≥ 3×]
[负面约束（结构描述后、悬浮词前）]
[悬浮/背景]
```

每段独立成句，每段写明**位置+材料+尺寸+可见性**。Lovart Nano Banana Pro 按"分件规格"来画图，没有分件描述它会自动用"品类默认"补全（参见 pitfall-025）。
