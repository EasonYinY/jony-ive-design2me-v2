# 用户要求显性化拆解方法

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 用户反馈"当用户有明确风格要求时，需要显性化表达，拆解为可执行的设计方向定义"

---

## 概述

当用户在brief中提出明确的风格/感觉/调性要求（如"科技感""简洁风""复古感""未来感""温暖感"等）时，**禁止**简单带过或仅作为标签粘贴。必须在STEP-03（行为与关系重写）阶段显性拆解为可执行的设计方向定义。

**核心原则**: 用户的风格词不是装饰，而是设计决策的输入。必须回答：**这个风格如何通过形态、材料、工艺、视觉语言来达成？**

---

## 拆解框架

### 拆解维度

| 维度 | 问题 | 示例（科技感） |
|------|------|----------------|
| **形态特征** | 什么形态能表达这个风格？ | 修长悬浮、非对称动态、极端比例 |
| **材料选择** | 什么材料能强化这个风格？ | 阳极氧化铝、碳纤维、镜面不锈钢 |
| **工艺表达** | 什么工艺能呈现这个风格？ | CNC精密加工、微喷砂、激光蚀刻 |
| **视觉语言** | 什么视觉元素能传达这个风格？ | 冷色调、悬浮感、精确间隙、LED微光 |
| **触感暗示** | 什么触感能强化这个风格？ | 冷滑、精准、无摩擦 |
| **交互反馈** | 什么交互能体现这个风格？ | 磁吸微震、触控条、渐变灯光 |

### 拆解示例

**用户要求**: "科技感"

❌ **错误做法**（标签粘贴）:
> 关系命题：电动剃须刀 = 手-面部接触 × 每日晨间仪式 × 科技感

✅ **正确做法**（显性拆解）:
> **科技感设计方向定义**:
> - **形态特征**: 修长悬浮感，刀头与手柄间2mm磁悬浮间隙，创造"未来科技"视觉奇观
> - **材料选择**: 阳极氧化铝（冷滑精准）+ 镍镀层镜面（反射科技感）
> - **工艺表达**: CNC一体成型（精密制造）+ 微喷砂（哑光科技感）
> - **视觉语言**: 冷银色调 + 悬浮间隙 + 侧面发丝级LED
> - **触感暗示**: 冷滑、无摩擦、磁吸缓冲
> - **交互反馈**: 拇指滑动触控条 → 磁吸微震 + LED渐亮

|--------|----------|----------|----------|----------|----------|
| **科技感** | 修长悬浮、非对称、极端比例 | 铝、碳纤维、镜面不锈钢 | CNC、微喷砂、激光蚀刻 | 冷色调、悬浮间隙、LED微光 | 冷滑、精准、无摩擦 |

## 应用规则

1. **必须在STEP-03完成拆解**: 风格词拆解是关系重写的一部分，不是后续补充
2. **每个风格词至少拆解3个维度**: 形态+材料+视觉语言为最小集合
3. **拆解必须与物理事实一致**: 不能为了风格而违反STEP-04的物理事实
4. **拆解必须可验证**: 每个拆解项都能在最终提示词中找到对应
5. **多个风格词需优先级排序**: 如"科技感+简洁风"，确定主次关系

## 检查清单

- [ ] 用户brief中是否包含明确的风格/感觉/调性词？
- [ ] 是否已拆解为形态+材料+工艺+视觉语言+触感+交互6个维度？
- [ ] 每个维度是否有具体可执行的定义（不是形容词堆砌）？
- [ ] 拆解是否与物理事实一致？
- [ ] 拆解是否能在最终提示词中验证？
- [ ] 多个风格词是否已确定优先级？
