## 子类型设计要点

### 1. 露营灯（便携照明）

**核心挑战**: 防水+光路+便携
**最佳形态**: 鹅卵石/水滴/胶囊
**材料推荐**: 硅胶+玻璃+金属+塑料（4材料对比）
**人体工学**: 单手握持，重量<500g
**关键策略**: 自然形态+多材料对比+便携性

**设计要点**:
- 形态: 鹅卵石/水滴（有机曲线，贴合手掌）
- 材料: 硅胶包覆（防滑）+ 磨砂玻璃（光扩散）+ 不锈钢底座（配重）
- 功能: 360°光扩散，调光，悬挂/放置双模式
- 防水: IPX4-IPX7

### 2. 壁灯（固定照明）

**核心挑战**: 安装+光扩散+建筑融合
**最佳形态**: 贝壳/水滴/叶片
**材料推荐**: 聚碳酸酯+铝/不锈钢（2-3材料）
**人体工学**: 安装高度1.8-2.5m，单手安装
**关键策略**: 文化杂交+建筑融合+材料对比

**设计要点**:
- 形态: 贝壳（开口光扩散，文化杂交）
- 材料: 聚碳酸酯（主体）+ 阳极氧化铝（边框）
- 功能: 定向/半定向光扩散，触摸开关
- 安装: 隐藏式安装，快速拆卸

### 3. 庭院灯（景观照明）

**核心挑战**: 氛围+光色+耐久
**最佳形态**: 蘑菇/竹节/禅石
**材料推荐**: 聚碳酸酯+不锈钢+硅胶（3材料）
**人体工学**: 地面放置，无需操作
**关键策略**: 氛围光+文化杂交+自然形态

**设计要点**:
- 形态: 蘑菇（帽状光扩散，氛围光）
- 材料: 磨砂聚碳酸酯（帽）+ 不锈钢（柄）+ 硅胶（底座）
- 功能: 氛围光，暖色调，柔和扩散
- 耐候: 抗UV，抗腐蚀

### 4. 头灯（穿戴照明）

**核心挑战**: 人体工学+重量+光路
**最佳形态**: 贝壳/水滴/叶片
**材料推荐**: 硅胶+铝+聚碳酸酯+织物（4材料）
**人体工学**: 额头贴合，重量<100g，防滑
**关键策略**: 人体工学优先+文化杂交+轻量化

**设计要点**:
- 形态: 贝壳（贴合额头曲线）
- 材料: 液态硅胶（贴合）+ 不锈钢（边框）+ 聚碳酸酯（发光面）+ 弹性织物（头带）
- 功能: 定向光扩散，调光，舒适佩戴
- 重量: <100g，重量分布均匀

### 5. 路灯（公共照明）

**核心挑战**: 高度+光覆盖+维护
**最佳形态**: 蘑菇/竹节/树干
**材料推荐**: 不锈钢+混凝土+玻璃（3材料）
**人体工学**: 公共使用，维护友好
**关键策略**: 比例张力+文化杂交+维护友好

**设计要点**:
- 形态: 蘑菇（帽柄比例，帽状光扩散）
- 材料: 不锈钢（帽）+ 混凝土（柄/底座）+ 磨砂玻璃（边缘）
- 功能: 大范围光覆盖，帽状扩散，维护友好
- 比例: 帽柄比例张力，视觉识别性

## 品类通用设计原则

### 形态选择

| 子类型 | 推荐形态 | 避免形态 | 理由 |
|--------|---------|---------|------|
| 露营灯 | 鹅卵石/水滴/胶囊 | 圆柱/球形 | 俗套形态 |
| 壁灯 | 贝壳/水滴/叶片 | 方形盒/圆柱 | 俗套形态 |
| 庭院灯 | 蘑菇/竹节/禅石 | 球形/圆柱 | 俗套形态 |
| 头灯 | 贝壳/水滴/叶片 | 传统头带式 | 俗套形态 |
| 路灯 | 蘑菇/竹节/树干 | 圆柱/传统灯柱 | 俗套形态 |

### 材料选择

| 子类型 | 推荐材料组合 | 材料数量 | 关键材料 |
|--------|-------------|---------|---------|
| 露营灯 | 硅胶+玻璃+金属+塑料 | 4种 | 硅胶（防滑） |
| 壁灯 | 聚碳酸酯+铝/不锈钢 | 2-3种 | 铝（边框） |
| 庭院灯 | 聚碳酸酯+不锈钢+硅胶 | 3种 | 硅胶（底座） |
| 头灯 | 硅胶+铝+聚碳酸酯+织物 | 4种 | 硅胶（贴合） |
| 路灯 | 不锈钢+混凝土+玻璃 | 3种 | 混凝土（结构） |

### 文化杂交建议

| 子类型 | 文化杂交方向 | 示例 |
|--------|-------------|------|
| 露营灯 | 自然×电子 | 河卵石×LED照明 |
| 壁灯 | 历史×现代 | 贝壳×建筑照明 |
| 庭院灯 | 生物×科技 | 蘑菇×氛围光 |
| 头灯 | 自然×人体工学 | 贝壳×额头贴合 |
| 路灯 | 自然×公共设计 | 蘑菇×北欧公共设计 |

## 检查清单

### 通用检查

```
[ ] 是否避免了品类俗套形态？
[ ] 是否使用了自然形态？
[ ] 是否包含文化杂交元素？
[ ] 是否使用了≥2种材料？
[ ] 是否考虑了人体工学？
[ ] 是否考虑了氛围光（景观照明）？
```

### 子类型特定检查

**露营灯**:
```
[ ] 重量<500g？
[ ] 防水IPX4+？
[ ] 360°光扩散？
[ ] 悬挂/放置双模式？
```

**壁灯**:
```
[ ] 安装高度1.8-2.5m？
[ ] 建筑融合？
[ ] 隐藏式安装？
[ ] 易拆卸维护？
```

**庭院灯**:
```
[ ] 氛围光设计？
[ ] 暖色调？
[ ] 耐候材料？
[ ] 景观融合？
```

**头灯**:
```
[ ] 重量<100g？
[ ] 额头贴合？
[ ] 防滑材料？
[ ] 重量分布均匀？
```

**路灯**:
```
[ ] 高度3-6m？
[ ] 大范围光覆盖？
[ ] 维护友好？
[ ] 比例张力？
```
