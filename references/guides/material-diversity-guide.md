## 代理差异化分配

| 代理 | 独特材料 | 材料组合 | 适用方向 |
|------|---------|---------|---------|
| Agent 1 | 氧化锆陶瓷 | 玻璃+铝+硅胶+陶瓷 | 禅石/鹅卵石 |

## 检查清单

### 设计前检查

```
[ ] 每个代理是否有至少1种独特材料？
[ ] 材料组合是否与方向形态匹配？
[ ] 材料对比是否清晰可见？
[ ] 材料职责是否明确？
```

### 设计后检查

```
[ ] 3方向的材料是否本质不同？
[ ] 材料对比是否增强了识别性？
[ ] 材料质感是否可描述？
[ ] 预测评分是否≥80/90？
```
