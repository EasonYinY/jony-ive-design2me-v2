# Pitfall 034: 方向同质化陷阱

> **版本**: 1.0
> **日期**: 2026-06-22
> **触发**: Round 2发现5个代理的设计方向高度相似
> **严重级别**: 🔴 高

---

## 问题描述

当多个代理遵循相同的设计规则时，导致方向同质化，六轴差异门在操作、材料、工艺维度上无法通过。

**Round 2实证**:
- 5个代理均使用"玻璃+铝+硅胶+陶瓷"4材料
- 5个代理均使用"触摸调光"操作
- 5个代理均使用"磨砂玻璃"工艺
- 3个代理选择了"禅石"形态

**关键发现**: 规则过于严格导致代理过度收敛

---

## 同质化类型

### 材料同质化

**现象**: 所有方向使用相同材料组合
**规避**: 每个代理至少使用1种独特材料

### 操作同质化

**现象**: 所有方向使用相同操作方式
**规避**: 每个代理必须使用不同的操作类型

### 工艺同质化

**现象**: 所有方向使用相同表面处理
**规避**: 每个代理必须使用不同的工艺类型

---

## 规避方法

### 方法1: 代理差异化矩阵

| 代理 | 操作维度 | 独特材料 | 独特工艺 |
|------|---------|---------|---------|
| Agent 1 | 触摸 | 氧化锆陶瓷 | 磨砂玻璃 |
| Agent 2 | 旋转 | 聚碳酸酯 | 光学级透明 |
| Agent 3 | 滑动 | 钛合金 | 拉丝纹理 |
| Agent 4 | 按压 | 碳纤维 | 碳纤维编织 |
| Agent 5 | 握持 | 木材 | 自然纹理 |

### 方法2: 六轴差异门强制检查

在STEP-09中增加强制检查:
- [ ] 操作维度是否不同？
- [ ] 材料维度是否不同？
- [ ] 工艺维度是否不同？

### 方法3: 随机化策略

每个代理在STEP-06候选生成时，随机选择:
- 1种操作类型
- 1种独特材料
- 1种独特工艺

---

## 检查清单

### 设计前检查

```
[ ] 每个代理的操作类型是否不同？
[ ] 每个代理是否有至少1种独特材料？
[ ] 每个代理是否有至少1种独特工艺？
[ ] 六轴差异门是否在所有维度上通过？
```

### 设计后检查

```
[ ] 3方向的操作是否本质不同？
[ ] 3方向的材料是否本质不同？
[ ] 3方向的工艺是否本质不同？
[ ] 预测评分是否≥80/90？
```
