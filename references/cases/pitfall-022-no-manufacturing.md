# Pitfall 022 - 缺少制造工艺考虑

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 用户反馈"没有按照加工工艺的形式生成，产品是落地量产需要分件，符合加工量产工艺"
> **严重级别**: MAJOR

---

## 触发条件

1. STEP-12 CMF 系统推导时，未标注分件方式
2. 未标注主要加工工艺
3. 使用「一体成型」等模糊描述
4. 形态超出常规加工工艺能力（如内部倒扣、零拔模角）

## 失败表现

- 产品像「整体雕塑」，无法拆解
- 形态无法制造（如注塑内部倒扣）
- 无分件线，无法装配
- 材料与工艺不匹配（如玻璃 CNC）
- 无法维修、无法更换刀头

## 根因分析

1. **分件设计缺失**: STEP-12 未强制要求分件方式
2. **工艺标注缺失**: 未要求每个部件标注加工工艺
3. **工艺约束缺失**: 未考虑注塑/CNC/压铸等工艺的实际约束
4. **制造知识缺失**: 设计人员缺乏制造工艺知识

## 修复方法

1. **分件方式**: 每个方向必须明确分件方式（上下壳/前后盖/模块化等）
2. **工艺标注**: 每个主要部件必须标注加工工艺
3. **工艺约束**: 遵守注塑/CNC/压铸等工艺约束
4. **分件线描述**: 提示词中必须显性描述分件线

## 验证方式

检查清单：
- [ ] 是否有明确的分件方式？
- [ ] 每个主要部件是否有明确的加工工艺？
- [ ] 加工工艺是否与材料匹配？
- [ ] 加工工艺是否与形态匹配？
- [ ] 分件线是否在提示词中显性描述？
- [ ] 产品是否可装配、可拆卸、可维修？
