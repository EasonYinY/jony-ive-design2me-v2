# Pitfall 039: 评审标准虚高陷阱

> **版本**: 1.0
> **日期**: 2026-06-21
> **触发**: 用户反馈"设计太飘逸、质感不顶级、缺少细节，评分降一半"
> **严重级别**: 🔴 高

---

## 问题描述

评审时使用"普通标准"会导致评分虚高，无法真实反映设计水平：

| 标准类型 | 评分示例 | 问题 |
|---------|---------|------|
| **普通标准** | 82/90 | 过于宽松，忽略AI生成缺陷 |
| **严格标准** | 81/90 | 略严格，但仍未触及本质 |
| **真实标准** | **4.35/10** | 反映真实水平，识别AI天花板 |

**关键洞察**：用户说"评分降一半"不是字面要求，而是要求**更严格的评审标准**。

---

## 实证数据

### V3产品级版评审对比

| 评审标准 | 评分 | 核心差异 |
|---------|------|---------|
| 普通标准（V1） | 82/90 | 忽略材料质感缺陷 |
| 严格标准（V2） | 81/90 | 略严格，但仍宽容 |
| 严格标准（V3） | 83/90 | 微升，但未触及本质 |
| **真实标准（V3）** | **4.35/10** | **识别AI天花板，诚实报告** |

### 图片质量分析数据

| 指标 | 数值 | 真实评估 |
|------|------|---------|
| 分辨率 | 2048×1536 | ✅ 高分辨率 |
| 边缘清晰度 | 7.92 (平均) | ❌ 边缘模糊，形态不清晰 |
| 材料质感 | 塑料感 | ❌ 6061-T6铝合金被理解为灰色塑料 |
| 功能细节 | 完全缺失 | ❌ USB-C、指示灯不可见 |
| 制造精度 | 模糊纹理 | ❌ CNC刀纹、0.1mm间隙不可见 |

---

## 虚高评分的危害

1. **误导用户**：让用户误以为设计已达艾维级
2. **阻碍改进**：虚高评分掩盖真实问题，无法针对性优化
3. **浪费迭代**：在错误的标准下迭代，无法真正提升
4. **失去信任**：用户发现评分虚高后，对评审失去信任

---

## 真实评审标准

### 8维度真实评审（严格版）

| 维度 | 真实标准 | 普通标准 | 差距 |
|------|---------|---------|------|
| 形态比例 | 边缘必须清晰，体量感明确 | 形态大致正确即可 | -3分 |
| 材料质感 | 材料必须真实，不能是塑料感 | 材料颜色正确即可 | -5分 |
| 接缝秩序 | 精密接缝必须可见 | 接缝存在即可 | -6分 |
| 人体工学 | 重量感、防滑纹理必须可见 | 尺寸大致合理即可 | -3分 |
| 识别性 | 功能细节必须清晰可见 | 形态独特即可 | -4分 |
| 静息状态 | 悬浮感、支撑必须可见 | 放置稳定即可 | -3分 |
| 背景纯净 | 纯白，无干扰 | 大致干净即可 | 0分 |
| 艾维对比 | 必须像苹果产品，不能是玩具 | 有设计感即可 | -6分 |

### 评分对照表

| 真实评分 | 对应普通评分 | 等级 | 说明 |
|---------|-------------|------|------|
| 9-10/10 | 95-100/90 | 超越艾维 | 极少数 |
| 8-9/10 | 85-95/90 | 艾维级 | 目标 |
| 7-8/10 | 75-85/90 | 准艾维级 | 可接受 |
| 6-7/10 | 65-75/90 | 普通级 | 需改进 |
| 5-6/10 | 55-65/90 | 低级 | 需大幅改进 |
| 4-5/10 | 45-55/90 | 不合格 | 重新设计 |

## 真实评审检查清单

### 评审前

```
[ ] 是否了解当前AI的能力天花板？
[ ] 是否使用真实标准而非普通标准？
[ ] 是否准备好诚实报告真实水平？
[ ] 是否准备好提供突破方案？
```

### 评审中

```
[ ] 形态边缘是否清晰？（不是模糊的概念图）
[ ] 材料质感是否真实？（不是塑料感）
[ ] 功能细节是否可见？（USB-C、指示灯、散热孔）
[ ] 制造精度是否表达？（CNC刀纹、精密接缝）
[ ] 重量感是否明确？（不是轻飘飘）
[ ] 是否像苹果产品？（不是玩具）
```

### 评审后

```
[ ] 评分是否真实反映水平？
[ ] 是否识别了AI天花板？
[ ] 是否提供了突破方案？
[ ] 用户是否接受真实评分？
```

---

## 用户沟通策略

当用户要求"评分降一半"时：

1. **不要字面理解**：不是数学计算，而是要求更严格标准
2. **诚实报告**：当前AI真实水平约4-5/10
3. **解释原因**：AI无法达到产品级精度
4. **提供方案**：更高质量模型/ControlNet/3D渲染
5. **设定预期**：分阶段目标（概念→细节→精确→产品级）
