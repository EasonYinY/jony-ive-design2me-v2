# Pitfall 014 - 颜色词替代高级感

## 触发条件

1. 使用"深岩灰""香槟金""砂岩白""深空灰"等颜色词作为设计高级感的主要来源
2. 颜色词替代材料职责说明（如"深岩灰液态硅胶"而非"液态硅胶，防滑减震，3年寿命"）
3. 颜色词替代工艺说明（如"香槟金阳极氧化"而非"阳极氧化铝，CNC铣切，10年寿命"）
4. 高级感被理解为"颜色+材质名称"而非"材料职责+工艺可见+比例精确+细节有目的"

## 失败表现

- 材料描述=颜色词+材料名称（如"深岩灰液态硅胶包覆不锈钢骨架"）
- 无法解释"为什么这个颜色？""颜色与功能有什么关系？"
- 图片中的材料呈现为"颜色块"（AI生成平滑色块，无真实纹理和工艺特征）
- 设计缺乏Ive式高级感：材料不诚实、工艺不可见、比例不精确、细节无目的

## 根因分析

### 直接原因
混淆了"视觉高级感"与"材料诚实"：
- 错误：高级感 = 颜色词（"深岩灰""香槟金"）+ 材料名称（"硅胶""铝"）
- 正确：高级感 = 材料职责明确 + 工艺可见 + 比例精确 + 细节有目的

### 深层原因
违反MTH-005"让材料承担职责"：
- 颜色词被当作设计决策依据，而非材料职责确定后的补充描述
- 工艺特征（如CNC纹理、注塑痕迹、喷砂质感）被颜色词掩盖

## 修复方法

### 高级感四支柱
必须同时满足：
1. **材料职责明确**：每个材料写明结构/触感/制造/寿命/维护职责
2. **工艺可见**：制造方式（CNC/注塑/铸造）和表面特征（纹理/光泽/接缝）诚实表达
3. **比例精确**：每个尺寸有工程或人体依据
4. **细节有目的**：每个倒角、接缝、纹理服务于功能或触感

### 颜色词降级规则
颜色词从"设计决策依据"降级为"视觉补充描述"：
1. 先确定材料名称、结构职责、触感职责、制造方式、寿命、维护方式
2. 再确定工艺特征（喷砂/拉丝/抛光/阳极氧化）
3. 最后才选择颜色词（且颜色词必须与工艺兼容，如"阳极氧化→space gray""喷砂→titanium natural"）

### 禁止事项
- ❌ "深岩灰液态硅胶"（颜色词替代材料职责）
- ❌ "香槟金阳极氧化铝"（颜色词替代工艺说明）
- ❌ "砂岩白ABS-PC合金"（颜色词替代材料性能说明）
- ✅ "液态硅胶，防滑减震，温润触感，液态注塑，3年寿命，可拆卸更换，颜色：深岩灰（与浴室环境协调）"

## 验证方式

1. 检查材料描述是否包含6个职责字段（材料名称|结构|触感|制造|寿命|维护）
2. 检查颜色词是否单独使用（无材料职责说明）
3. 检查图片中的材料是否呈现真实纹理和工艺特征（非平滑色块）
4. 检查设计是否满足高级感四支柱（材料职责+工艺可见+比例精确+细节有目的）
