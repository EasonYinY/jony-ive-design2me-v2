# Rule 013 - 制造工艺硬约束

## 规则定义

每个设计方向必须明确标注分件方式和主要加工工艺，禁止「整体雕塑」式不可制造设计。

## 适用范围

STEP-12 CMF 系统推导、STEP-13 DesignIR、STEP-14 审美质量门

## 执行标准

### 强制要求

1. **每个方向必须明确分件方式**
   - 分件方式：上下壳 / 前后盖 / 模块化 / 套筒式 / 卡扣式 / 螺丝固定
   - 分件线必须在形态上可见或隐藏
   - 分件方式必须与功能一致

2. **每个主要部件必须标注加工工艺**
   - 工艺类型：CNC 五轴 / 注塑成型 / 压铸 / 3D 打印 / 冲压 / 吹塑 / 电铸 / 微蚀刻
   - 加工工艺必须与材料匹配
   - 加工工艺必须与形态匹配

3. **分件线必须在提示词中显性描述**
   - 描述内容：分件位置、分件方式、材料差异、工艺差异
   - 禁止「一体成型」等模糊描述

4. **量产工艺约束必须遵守**
   - 注塑：壁厚均匀、拔模角、无倒扣
   - CNC：刀具可达性、最小圆角
   - 压铸：壁厚均匀、拔模角

### 禁止行为

- ❌ 「一体成型」等模糊描述
- ❌ 无分件线的产品设计
- ❌ 加工工艺与材料不匹配
- ❌ 加工工艺与形态不匹配（如注塑内部倒扣）

## 违反后果

STEP-12 CMF 系统推导不通过，退回重新推导。

## 验证方式

检查清单：
- [ ] 是否有明确的分件方式？
- [ ] 每个主要部件是否有明确的加工工艺？
- [ ] 加工工艺是否与材料匹配？
- [ ] 加工工艺是否与形态匹配？
- [ ] 分件线是否在提示词中显性描述？
- [ ] 产品是否可装配、可拆卸、可维修？
