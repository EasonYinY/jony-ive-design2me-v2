# Rule 011 - CMF 软硬对比硬规则

## 规则定义

每个设计方向必须包含「硬」与「柔」的对比，且「柔」的部分必须有明确规律。

## 适用范围

STEP-12 CMF 系统推导、STEP-14 审美质量门

## 执行标准

### 强制要求

1. **每个方向必须有 ≥1 个硬元素 + ≥1 个柔元素**
   - 硬元素：直线、锐角、平面、金属/玻璃/陶瓷
   - 柔元素：曲线、圆角、曲面、橡胶/硅胶/液态金属涂层
   - 硬与柔必须在同一产品中并存

2. **柔元素必须有明确规律**
   - 规律类型：G2 连续曲率 / 等距纹理 / 渐变曲率 / 对称流动 / 螺旋规律
   - 规律必须在提示词中描述

3. **软硬交接处必须有明确过渡区**
   - 过渡类型：G2 圆角过渡 / 渐消面 / 咬合过渡 / 层叠过渡
   - 过渡区必须可见、可触、可描述

4. **液态金属特殊规则**
   - 不得覆盖产品整体 >50% 面积
   - 必须有明确的流动方向和形态变化
   - 与硬材料的边界必须清晰

### 禁止行为

- ❌ 全柔设计（无硬元素支撑）
- ❌ 全硬设计（无柔元素缓冲）
- ❌ 「随意流动」「有机曲线」等无规律描述
- ❌ 液态金属整体覆盖

## 违反后果

STEP-12 CMF 系统推导不通过，退回重新推导。

## 验证方式

检查清单：
- [ ] 是否有 ≥1 个硬元素？
- [ ] 是否有 ≥1 个柔元素？
- [ ] 柔元素是否有明确规律？
- [ ] 软硬交接处是否有过渡区？
- [ ] 液态金属是否局部应用？
- [ ] 液态金属是否有形态变化？

## 来源

- 用户 2026-06-21 反馈："液态金属造型有些单一，没有一些形态变化，有柔的，那你要考虑硬的地方。然后柔的东西它是怎么个柔法？它也需要有一定的规律规则"
