# 技术克制原则

## 核心规则

1. **高技术上限**：每个方案最多使用1项"高技术"（磁悬浮/压电/全息/AI/量子等）
2. **技术-概念匹配**：高技术必须服务于核心概念，而非反向（不能为了用磁悬浮而设计磁悬浮）
3. **低技术高概念优先**：优先使用成熟技术（机械/材料/形态）创造概念性设计
4. **技术必要性检查**：每项技术必须回答"不用这项技术，核心概念是否成立？"若答案是"是"，则删除该技术

## 高技术定义

| 技术 | 等级 | 说明 |
|------|------|------|
| 磁悬浮/磁吸 | 高技术 | 需要磁体+控制电路 |
| 压电/触觉反馈 | 高技术 | 需要压电材料+驱动电路 |
| 全息/AR显示 | 高技术 | 需要光学系统+计算 |
| AI/自适应 | 高技术 | 需要传感器+算法 |
| LED/灯光 | 中技术 | 简单电路，但需克制使用 |
| 机械结构 | 低技术 | 铰链/弹簧/卡扣 |
| 材料创新 | 低技术 | 新材料应用，非新发明 |

## 技术使用检查清单

- [ ] 本方案是否超过1项高技术？
- [ ] 每项高技术是否服务于核心概念？
- [ ] 去掉高技术后，核心概念是否仍然成立？
- [ ] 是否有低技术方案可以达到同样效果？
- [ ] 技术是否增加了不必要的复杂性？

## 违规处理

若方案违反技术克制原则：
1. 删除多余高技术，保留最核心1项
2. 用低技术替代（机械/材料/形态创新）
3. 重新验证概念是否仍然成立
