# 审美收敛质量门

> **版本**: 1.1.0
> **日期**: 2026-06-21
> **触发**: 优秀提示词分析发现视觉语言一致性是关键

---

## 质量门检查项

分别检查产品识别性、比例张力、部件层级、材料诚实、触觉精度、接缝秩序、静息状态和十年耐久。八项必须各有一条 `AestheticCriterionAssessment`，不得缺失或重复。

**第九项：识别性设计**（标志性形态、标志性细节、材料对比、比例记忆），参考 `references/guides/identity-design.md`。识别性评分需 ≥ 8/10。

**第十项：视觉语言一致性**，参考下方「视觉语言一致性检查」。

每项判断必须包含具体结论、`design_ir.<field>` 或 `image.<evidence>` 证据指针和通过状态，不使用“不够高级”之类无对照结论。只要一项失败，结果即为 `RETURN_TO_DIRECTION_DEVELOPMENT`；**十项**全部通过才返回 `PASS`，摄影和提示词不能覆盖失败。

本质量门是工程治理规则，不冒充 Jony Ive 的公开理论，也不允许用人物风格锚点替代具体判断。

---

## 视觉语言一致性检查

### 检查目的

确保三个方向的提示词共享同一视觉语言基础，形成 LoveFrom 级别的统一视觉识别。

### 检查项

| 检查项 | 通过标准 | 失败标准 |
|--------|---------|---------|
| **悬浮感一致性** | 三个方向都使用 "Floats" 或等效悬浮描述 | 一个方向落地，其他悬浮 |
| **背景一致性** | 三个方向都使用 "neutral gradient" 或 "pure white" | 一个方向有场景背景 |
| **焦点一致性** | 三个方向都只包含产品，无环境元素 | 一个方向包含人物/辅助物品 |
| **摄影语法一致性** | 三个方向使用相同的摄影描述结构 | 摄影描述结构混乱不一致 |
| **负面约束一致性** | 三个方向都使用 "No logo, no text..." 结构 | 负面约束格式不统一 |
| **材料精确度** | 三个方向都使用精确材料名称+结构状态 | 一个方向使用模糊材料词 |
| **几何精确度** | 三个方向都包含至少2个精确数字 | 一个方向无精确数字 |

### 评分标准

| 等级 | 分数 | 描述 |
|------|------|------|
| 优秀 | 7/7 | 全部通过，视觉语言高度统一 |
| 良好 | 5-6/7 | 基本通过， minor inconsistency |
| 待改进 | 3-4/7 | 部分通过，需要调整 |
| 失败 | <3/7 | 视觉语言混乱，需要重新编译 |

### 修复路径

视觉语言一致性失败时：
1. 检查 `photography-controls.md` 的封闭枚举是否被正确执行
2. 检查 `prompt-compression.md` 的强制保留字段是否被删除
3. 重新编译三个方向的提示词，确保统一视觉语言
4. 必要时退回 `STEP-12` 重新检查 CMF 系统的一致性

## 第十一项：基础形态锁定检查

### 检查目的

确保形态不是从"有机曲线""未来感"等抽象词开始，而是先定义基础体块，再写变形规则。

### 检查项

| 检查项 | 通过标准 | 失败标准 |
|--------|---------|---------|
| **基础体块定义** | 提示词先定义基础形态（slab/block/cylinder/dome/wedge/disc） | 直接从"有机曲线""流线型"开始 |
| **精确尺寸** | 包含至少2个精确数字（尺寸/角度/曲率） | 无精确数字或只有1个 |
| **几何术语** | 使用至少1个几何术语（G2/G1/fillet/chamfer/tangent/step） | 无几何术语 |
| **变形依据** | 变形有明确依据（人体工学/功能/物理） | 变形无依据，随意 |
| **单材质主导** | 1种主导材料（80%视觉面积），辅助≤1种 | 3种以上材料并列 |
| **姿态锁定** | 包含姿态词（Floats/levitates/rests on/suspended/nestles） | 无姿态词或落地描述 |
| **负面约束精简** | 3-5个具体负面约束 | <3个或>5个 |

### 评分标准

| 等级 | 分数 | 描述 |
|------|------|------|
| 优秀 | 7/7 | 全部通过，基础形态锁定完整 |
| 良好 | 5-6/7 | 基本通过，minor issue |
| 待改进 | 3-4/7 | 部分通过，需要调整 |
| 失败 | <3/7 | 基础形态缺失，需要重新推导 |

### 修复路径

基础形态锁定失败时：
1. 退回 STEP-10，重新选择基础体块
2. 定义精确尺寸（长×宽×高）
3. 定义变形规则（必须有理性依据）
4. 添加几何术语和精确数字
5. 确保单材质主导和姿态锁定

## 第十二项：原型拓扑检查（Archetype Topology Check）

### 检查目的

确保复杂工业系统的设计保留了品类原型特征，防止形态分化导致产品无法识别。

### 检查项

| 检查项 | 通过标准 | 失败标准 |
|--------|---------|---------|
| **品类锚定词** | 提示词第一段包含明确的品类锚定词（如 MRI scanner / espresso machine / drone） | 纯几何体替代（如 slab / cube / cylinder） |
| **不可缩减特征** | 三大形态特征至少保留两个，且视觉权重 > 40% | 特征被缩减或省略 |
| **功能暗示词** | 提示词包含至少一个功能暗示词（如 patient cradle / scanning bore / brew group） | 无功能暗示词 |
| **尺度锚定** | 包含显性尺寸和人体参照 | 无尺寸或尺度漂移 |
| **材料诚实性** | 重型设备使用工程复合材料/高刚性合金/医用级高分子 | 使用建筑/家具材料（travertine/walnut/marble） |
| **细节加载** | 包含至少一个三级细节层（认知状态UI / 工程信任标签 / 制造工艺接口） | 产品表面"绝对虚无" |
| **禁止纯几何替代** | 几何体仅作为形态修饰，不作为语义主语 | 纯几何体完全替代品类词 |

### 评分标准

| 等级 | 分数 | 描述 |
|------|------|------|
| 优秀 | 7/7 | 全部通过，原型拓扑完整保留 |
| 良好 | 5-6/7 | 基本通过，minor issue |
| 待改进 | 3-4/7 | 部分通过，需要调整 |
| 失败 | <3/7 | 原型丢失，需要重新推导 |

### 修复路径

原型拓扑检查失败时：
1. 退回 STEP-05.7，重新运行行业原型锚定门
2. 检查 `archetype-verification.md` 的不可缩减三大特征
3. 重新编译提示词，确保品类锚定词和功能暗示词
4. 检查材料职责诚实性，替换建筑/家具材料
5. 添加至少一个三级细节层
