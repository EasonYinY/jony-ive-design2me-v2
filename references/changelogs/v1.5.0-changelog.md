# v1.5.0 Changelog

## 新增内容

### Pitfalls
- **Pitfall 029**: 复杂产品衰减增大 (complex-product-attenuation)
  - 现象: 复杂产品（音箱、咖啡机）比简单产品衰减更大
  - 根因: 复杂产品需要描述更多部件，机械结构占用提示词空间
  - 解决: 优先使用自然材料，减少机械结构描述，调整预期，分两次生成

### Guides
- **Guide 022**: 织物/碳纤维质感精确表达 (textile-carbon-fiber-expression)
  - 织物: 编织方式、线径、密度
  - 碳纤维: 编织 pattern、层数、表面处理

### 升级指南
- **Guide 019**: 动态形态表达（patch）
  - 新增"折叠状态指示器"章节
