# v1.3.0 Changelog

## 新增内容

### Pitfalls
- **Pitfall 027**: 交互细节衰减 (interaction-detail-attenuation)
  - 现象: 设计阶段交互评分 8-9/10 → 图片评审 5-7/10，衰减 1-2 分
  - 根因: 尺寸太小、对比度太低、位置不突出、弱化词使用
  - 解决: 尺寸放大2-3倍、对比度强化、位置突出、多线索叠加

### Guides
- **Guide 019**: 交互细节可视化增强 v2 (interaction-detail-visualization-v2)
  - 尺寸放大法: 交互元素尺寸放大2-3倍
  - 对比度强化法: 不同材料强对比
  - 位置突出法: 视觉焦点位置
  - 多线索叠加法: 尺寸+对比+位置+表面处理+发光

- **Guide 020**: 识别性设计增强 v2 (identity-design-v2)
  - 识别性锚点法: 极端到无法忘记的特征
  - 多层识别法: 远/中/近三层识别
  - 品类突破法: 至少一个维度突破品类认知

- **Guide 021**: 咖啡机品类交互设计指南 (coffee-machine-interaction)
  - 压力可视化: LED环颜色渐变
  - 萃取进度: 分段LED条
  - 温度指示: 色温变化
  - Crema观察窗: 透明窗口
  - 蒸汽/热水切换: 旋转切换环

- **Guide 022**: 动态形态图片表达指南 (dynamic-form-visualization)
  - 多状态暗示法: 部分展开暗示动态
  - 动态线条法: 螺旋线暗示旋转
  - 残影/轨迹法: 轻微模糊暗示运动
  - 多视角组合法: 单一图片多视角
  - 光影动态法: 光影方向暗示运动

## 升级触发

- **Round 2 产品**: 便携式咖啡机
- **评审结果**: 2/3方向艾维级（D1: 80/90, D3: 82/90），D2: 79/90
- **核心问题**: 交互细节衰减 1-2 分、D2识别性 8.5/10距艾维级差0.5分
- **升级目标**: 减少交互衰减至≤0.3分，提升识别性至≥9/10
