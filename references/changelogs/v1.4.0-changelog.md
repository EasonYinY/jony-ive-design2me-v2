# v1.4.0 Changelog

## 新增内容

### Pitfalls
- **Pitfall 028**: 自然形态表达不稳定 (natural-form-instability)
  - 现象: 鹅卵石等自然形态被AI理解为"不规则石头"
  - 根因: AI对"自然"理解偏向随机
  - 解决: 强调"受控的自然"，材质前置，添加设计修饰词

### Guides
- **Guide 020**: 静态图LED状态表达 (led-state-in-static-image)
  - 静态图无法展示LED发光状态的应对策略

- **Guide 021**: 透明材料表达 (transparent-material-expression)
  - 亚克力/玻璃透明度不足的解决方法
