# v1.2.0 Changelog

## 新增内容

### Pitfalls
- **Pitfall 026**: 提示词→图片衰减 (prompt-to-image-attenuation)
  - 现象: 设计阶段评分 → 图片评审评分平均衰减 0.77 分
  - 根因: 抽象概念AI理解有限、信息密度不足、模型语义偏差
  - 解决: 抽象概念→具体视觉转换、信息密度增强、分层提示词结构、验证循环

### Guides
- **Guide 015**: 多材料接缝设计指南 (multi-material-joint-design)
  - 艾维接缝处理原则: 完全隐藏或故意强调
  - 隐藏式/强调式接缝的提示词模板
  - 常见材料对的接缝策略表

- **Guide 016**: 材料诚实可视化指南 (material-honesty-visualization)
  - 艾维材料诚实三原则: 结构诚实、触感诚实、制造诚实
  - 3层表达法: 结构功能显性化、触感特征可辨识、制造痕迹合理
  - 常见违背类型和修正方法

- **Guide 017**: 文化杂交视觉化指南 (cultural-hybrid-visualization)
  - 文化杂交可视化公式: [文化A] × [文化B] = [具体视觉元素]
  - 日本/北欧/工程文化元素对照表
  - 3个修正版示例（光之桥/光之柱/光之翼）

- **Guide 018**: 交互细节可视化指南 (interaction-detail-visualization)
  - 交互细节可视化公式: [功能] = [位置] + [视觉线索] + [触觉线索] + [尺寸]
  - 高频接触点模板和示例
  - 交互节点模板和示例

## 升级触发

- **Round 1 产品**: 智能桌面台灯
- **评审结果**: 方向B（光之柱）8.45/10 — 准艾维级，距艾维级差 0.44 分
- **核心问题**: 提示词→图片衰减 0.77 分、接缝处理不足、材料诚实违背
- **升级目标**: 减少衰减至 0.3 分以内，提升图片评审至艾维级
