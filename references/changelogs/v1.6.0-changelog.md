# v1.6.0 Changelog

## 新增内容

### Pitfalls
- **Pitfall 030**: 佩戴状态不可见 (wearable-state-invisible)
  - 现象: 可穿戴产品佩戴状态在静态图中不可见
  - 根因: 静态图无法展示产品与人体的关系
  - 解决: 提示词描述佩戴状态，使用人体模型，接受限制

- **Pitfall 031**: 曲面屏幕透明度 (curved-screen-transparency)
  - 现象: 曲面蓝宝石/玻璃屏幕在AI生成中偏白
  - 根因: AI对"曲面+透明"理解偏差
  - 解决: 使用强透明词，描述折射，描述屏幕下内容

### Guides
- **Guide 023**: 可穿戴产品人体工学 (wearable-ergonomics-guide)
  - 手腕贴合、重量分布、表带设计、传感器位置
  - 系统化的可穿戴产品设计参数

### 关键洞察
1. **自然形态优势**: 鹅卵石(R3)、水滴(R5)评分最高
2. **产品复杂度**: 简单产品(充电器)更易达到艾维级
3. **材料选择**: 陶瓷、钛镜面抛光表达最稳定
4. **文化杂交**: 强文化符号(日本禅石、中国水墨)增强识别性
5. **衰减控制**: 300-450词提示词+自然材料=最小衰减
