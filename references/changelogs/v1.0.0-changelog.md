## 核心架构

- **四阶段十五步主流程**: Brief 解析 → 方向筛选 → 方向展开 → 表达与评审
- **五维决策引擎**: vision_value / strategy_system / tools_infrastructure / execution_tactics / environment_timing
- **DesignIR**: 产品中间表示合同（12 个必填字段）
- **八项审美质量门**: product_identity, proportion_tension, component_hierarchy, material_honesty, tactile_precision, seam_order, resting_state, ten_year_durability
- **三级图片修复逻辑**: PASS / EXPRESSION_REPAIR / RETURN_TO_PRODUCT_DERIVATION

## 关键设计决策

1. **默认在提示词处停止**: 不自动出图，需用户显式授权
2. **技能优化模式**: 用户反馈默认只优化技能，不重跑原项目
3. **三个方向必须本质不同**: 关系、结构、操作三个维度唯一
4. **证据等级体系**: S/A/B/C/D/G/U 替代 SECI 隐性知识模型
