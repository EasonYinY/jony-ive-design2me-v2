## 变更内容

### 协议层(永久有效)

**模型适配优先协议**(取代 v2.2.0 的"产品级革命"路线):
- 提示词结构对齐 Google Cloud 官方公式:`Subject + Action + Location + Composition + Style`,Camera/Lighting 在第 5 段(末尾)
- 数字精度硬约束改为 `≤ 2 个 ≥ 1mm 粗数字,0 个 < 1mm 精度数字`
- 提示词长度 sweet spot:40-80 words(ppl.studio 1000+ 张实测)
- 工艺词库 ≥ 30 种,3 方向强制互斥
- 颜色词库 + 光感词库 3 方向强制互斥(0 个 < 1mm 数字 + amber LED 等会强制 Nano Banana Pro 走训练集先验的词)
