## 审计报告核心发现

用户通过对 `3t-20260624.json` 设计会话的完整逆向剖析，发现核心病灶：**设计工作流在高度抽象的"哲学推导层"运行完美，但在向落地形态转译的"提示词工程层"与"模型适配层"发生了严重的"语义解离（Semantic Dissociation）"**。

两个底层病灶：
1. **行业原型（Archetype）的彻底丢失**：MRI 核心拓扑被泛化为"悬浮石板"或"分离双板"，模型失去类目锚点，误识别为建筑/家具
2. **功能的反自然虚无（Unnatural Emptiness）**：机械执行"No logo, no text"，剥离了工业产品必需的认知信任基础（丝印、警告标签、UI界面、操作刻度）

## 升级内容

### 1. 新增 STEP-05.7：行业原型锚定门（Archetype Verification Gate）

**文件**: `references/process/end-to-end-workflow.md`

在核心矛盾提炼后、候选生成前，强制注入行业原型拓扑保护协议：
- **拓扑完备性**：定义产品品类的【不可缩减三大形态特征】，确保视觉权重 > 40%
- **尺度错觉防御**：通过 `[净尺寸需求 + 结构厚度容差 + 安全防护边界 = 最终输出数值]` 硬校验
- **功能 Legibility**：提示词第一段必须包含品类锚定词和功能暗示词
- **禁止纯几何替代**：严禁用 slab/cube/cylinder 完全替代行业原型词

**新增参考文档**: `references/guides/archetype-verification.md` (REF-GUIDE-ARCHETYPE)

### 2. 新增 STEP-15.7：高秩序工业细节加载协议

**文件**: `references/process/end-to-end-workflow.md`

针对复杂工业系统，引入三级细节加载，将"克制"重新定义为"复杂被秩序收束"：
1. **认知状态指示层（Cognitive State UI）**：Flush-integrated micro-typographic control zones
2. **法律与工程信任标签（Regulatory & Engineering Veracity）**：Sub-millimeter laser-etched scale markings
3. **制造工艺接口（Manufacturing Interface Detail）**：0.3mm tight engineering gaps

**升级文件**: `references/guides/prompt-engineering.md`（新增方法24-26）

### 3. 新增 STEP-15.8：材料职责诚实性校验

**文件**: `references/process/end-to-end-workflow.md`

- 禁止将建筑/家具材料（travertine, walnut, marble）用于重型精密医疗设备
- 重型设备必须使用工程复合材料/高刚性合金/医用级高分子
- 材料必须承担功能职责（电磁屏蔽/结构刚性/热管理）

### 4. DesignIR 新增三个字段

**文件**: `references/design/design-ir.md`

- `archetype_features`：行业原型三大特征
- `detail_loading_protocol`：三级细节加载协议
- `material_responsibility`：材料职责诚实性

### 5. 审美质量门新增第十二项：原型拓扑检查

**文件**: `references/quality/aesthetic-gate.md`

新增 7 维度原型拓扑检查：
- 品类锚定词 / 不可缩减特征 / 功能暗示词 / 尺度锚定 / 材料诚实性 / 细节加载 / 禁止纯几何替代

## 第二次审计升级（2026-06-25）

### 6. 新增 STEP-15.9：五段式权重重组架构（Five-Layer Weighted Prompt Architecture）

**文件**: `references/process/end-to-end-workflow.md`

STEP-15 输出结构重组为五大独立层级，按固定权重比例组织：
- **Layer A (20%)**: 摄影环境与体量尺度控制 — 动态选择摄影语料（小型/中型/大型设备）
- **Layer B (30%)**: 拓扑原型与主几何秩序 — 锁死品类基本功能拓扑（Volume Weight > 40%）
- **Layer C (15%)**: 高秩序微观细节与丝印系统 — 至少一个三级细节层（方法24-26）
- **Layer D (20%)**: 尺度材料职责与时间老化 — [material] for [specific function] 三问闭环
- **Layer E (15%)**: 光影控制与复杂性负向排除 — 禁止一刀切否定词，改用"whispered rather than screamed"

**升级文件**: `references/guides/prompt-engineering.md`（新增方法27：五段式权重重组）

### 7. 新增 STEP-15.10：防御性决策树（Fail-Safe Verification Gate）

**文件**: `references/process/end-to-end-workflow.md`

最终输出前运行自动化审计分支逻辑：
- **规则1**: 绝对化否定词拦截（No text/logo → 强制替换为高秩序收束描述）
- **规则2**: 材质尺度错配拦截（travertine/walnut + 大型设备 → 强制替换为建筑级材质）
- **规则3**: 原型丢失拦截（纯几何替代品类词 → 强制打断并插入原型结构）

**新增参考文档**: `references/guides/fail-safe-verification-gate.md` (REF-GUIDE-FAILSAFE)

### 8. 新增方法28：尺度自适应 CMF 与摄影编译算法

**文件**: `references/guides/prompt-engineering.md`

根据产品物理尺寸（大/中/小三档）动态调用不同阶数的 CMF 语料与摄影语料：
- 小型（< 30cm）：桌面级材质 + Studio photograph macro
- 中型（30cm - 1.5m）：设备级材质 + Studio photograph medium
- 大型（> 1.5m）：建筑级/工业级材质 + Architectural-grade composition

**新增参考文档**: `references/guides/scale-aware-cmf-compiler.md` (REF-GUIDE-SCALE-CMF)

### 9. 新增方法29：防御性决策树（Fail-Safe Verification Gate）

**文件**: `references/guides/prompt-engineering.md`

在最终输出提示词前，运行三类自动化审计：
- 绝对化否定词漂移 → 高秩序收束描述
- 材质尺度错配漂移 → 建筑级材质替换
- 原型丢失漂移 → 强制插入原型结构
