# 摄影表达控制

摄影只控制机位、构图、光线、背景、景深和既有字段的可见度。产品英雄图优先使用能同时看清主轮廓、关键连接和材料边界的三分之四视角；使用图只有在动作需要验证时启用。

摄影参数不得创造构件、掩盖比例冲突或用光效伪造材料品质。
